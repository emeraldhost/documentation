# V Rising-Server Admin Hinzufügen

:::: info Info
Wenn Du mehrere Admins hinzufügen möchtest, füge einfach **mehrere Steam64IDs** ein - jede in einer **neuen Zeile**.
::::

:::: tip Tipp
Hier findest du eine Anleitung, wie du deine [Steam64ID](../steamid64-herausfinden.md) herausfindest.
::::

1. <b>Dashboard öffnen</b><br>
   Öffne Dein Dashboard und wähle Deinen V Rising Server aus.

2. <b>Ordner öffnen</b><br>
   Navigiere zum Ordner `save-data/Settings`.

3. <b>Datei öffnen</b><br>
   Öffne die Datei `adminlist.txt`.

4. <b>Steam64ID eintragen</b><br>
   Füge Deine Steam64ID in einer neuen Zeile ein.

5. <b>Server neustarten</b><br>
   Speichere die Datei und Starte Deinen Server neu, damit die Änderungen übernommen werden.
