# V Rising-Server Savegame Hinzufügen

:::: danger Achtung
Wenn Du ein neues Savegame hinzufügst, **wird Dein aktueller Spielstand überschrieben** und kann **nicht wiederhergestellt** werden.
Erstelle unbedingt ein **Backup**, falls Du Dein aktuelles Savegame später erneut verwenden möchtest.
::::

:::: info Info
Ein lokales Savegame findest du im folgenden Verzeichnis auf Deinem PC: ```C:\Users\<Benutzername>\AppData\LocalLow\Stunlock Studios\VRising\Saves```
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne Dein Dashboard und wähle Deinen V Rising Server aus.

2. <b>Server stoppen</b><br>
   Stoppe Deinen V Rising Server.

3. <b>Zum Ordner navigieren</b><br>
   Navigiere zum Ordner ```Saves/v1```.

4. <b>Savegame-Ordner öffnen</b><br>
   Öffne den Ordner Deines Savegames.

5. <b>Dateien hochladen</b><br>
   Lösche die vorhandenen Dateien und lade die neuen Dateien Deines gewünschten Savegames hoch.

6. <b>Server neu starten</b><br>
   Starte Deinen Server neu, damit die Änderungen übernommen werden.
