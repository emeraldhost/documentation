# So änderst du das Speicher-Intervall auf deinem V Rising Server

Du kannst einstellen, wie oft und wie viele automatische Speicherungen der Server erstellt.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Speicher-Einstellungen anpassen</b><br>
   Passe die folgenden Felder an:

   | Feld | Beschreibung |
   |------|-------------|
   | **Automatisches Speicherintervall** | Zeit in Sekunden zwischen automatischen Speicherungen |
   | **Anzahl der automatischen Speicherungen** | Wie viele Speicherstände aufbewahrt werden |

4. <b>Server neu starten</b><br>
   Speichere die Einstellungen und starte deinen Server neu.
