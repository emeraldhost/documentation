# So änderst du die Spieleinstellungen auf deinem V Rising Server

Du kannst die voreingestellten Spieleinstellungen über die Verwaltung anpassen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Spieleinstellungen anpassen</b><br>
   Wähle im Feld **Voreingestellte Spieleinstellungen** das gewünschte Preset aus.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
