# So aktivierst oder deaktivierst du die Serverliste auf deinem V Rising Server

Du kannst einstellen, ob dein Server in der Steam- und/oder Epic Games-Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Serverliste einstellen</b><br>
   Passe die folgenden Felder an:

   | Feld | Werte | Beschreibung |
   |------|-------|-------------|
   | **Steam Serverliste** | `true` / `false` | Server in der Steam-Serverliste anzeigen |
   | **Epic Games Serverliste** | `true` / `false` | Server in der Epic Games-Serverliste anzeigen |

4. <b>Server neu starten</b><br>
   Speichere die Einstellungen und starte deinen Server neu.
