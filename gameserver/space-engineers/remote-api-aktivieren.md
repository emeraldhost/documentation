# So aktivierst du die Remote API auf deinem Space Engineers Server

Die Remote API (VRage Remote API) ermöglicht die Fernverwaltung deines Servers über eine Schnittstelle — zum Beispiel, um den Serverstatus abzufragen oder den Server über externe Tools zu steuern.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Remote API aktivieren</b><br>
   Setze die Einstellung **Remote API** auf `true`, um sie zu aktivieren, oder auf `false`, um sie zu deaktivieren.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Der Port für die Remote API ist bereits automatisch gesetzt — du musst ihn nicht selbst konfigurieren.
::::
