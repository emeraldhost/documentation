# So änderst du die Server Beschreibung deines Space Engineers Servers

Die Server Beschreibung wird Spielern in der Serverliste zusätzlich zum Server Namen angezeigt.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Beschreibung ändern</b><br>
   Trage im Feld **Server Beschreibung** den gewünschten Text ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
