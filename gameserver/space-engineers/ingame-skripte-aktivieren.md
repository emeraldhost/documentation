# So erlaubst du Ingame Skripte auf deinem Space Engineers Server

Mit dieser Einstellung erlaubst du das Ausführen von Ingame-Skripten — den C#-Skripten in **Programmierbaren Blöcken**. Ohne diese Option werden solche Skripte beim Laden entfernt.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Ingame Skripte aktivieren</b><br>
   Setze die Einstellung **Ingame Skripte** auf `true`, um sie zu erlauben, oder auf `false`, um sie zu verbieten.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Ingame Skripte benötigen den aktivierten [Experimental Modus](experimental-modus-aktivieren.md) — aktiviere ihn zusätzlich, damit die Skripte ausgeführt werden.
::::

:::: warning Achtung
Ingame Skripte sind **keine Mods**: Sie werden nicht in die Mod-Liste eingetragen (siehe [Mods hinzufügen](mods-hinzufuegen.md)), sondern im Spiel direkt in einen Programmierbaren Block eingefügt.
::::
