# So änderst du den Game Mode deines Space Engineers Servers

Der Game Mode legt fest, in welchem Spielmodus dein Server läuft.

- **Survival** — Überlebensmodus mit Ressourcenmanagement, Energie, Sauerstoff und Fertigung.
- **Creative** — Kreativmodus mit unbegrenzten Ressourcen und freiem Bauen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Game Mode ändern</b><br>
   Wähle im Feld **Game Mode** den gewünschten Wert (`Survival` oder `Creative`) und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
