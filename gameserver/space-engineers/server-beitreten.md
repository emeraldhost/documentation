# So trittst du deinem Space Engineers Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die **IP-Adresse** und den **Port** deines Servers findest du in der **Verwaltung** deines Servers in der Übersicht. Verwende genau diese Adresse zum Verbinden.
::::

## Über Direktverbindung

1. <b>Space Engineers starten</b><br>
   Starte Space Engineers und wähle im Hauptmenü **Join Game**.

2. <b>Direktverbindung öffnen</b><br>
   Klicke auf **Direct Connect**. Falls ein „Quick Start"-Popup erscheint, wähle **No**.

3. <b>Adresse eingeben</b><br>
   Gib die IP-Adresse und den Port deines Servers ein:

   ```
   123.45.67.89:27016
   ```

4. <b>Verbinden</b><br>
   Klicke auf **Connect**, um dem Server beizutreten.

## Über den Server-Browser

1. <b>Space Engineers starten</b><br>
   Starte Space Engineers und wähle im Hauptmenü **Join Game**.

2. <b>Server suchen</b><br>
   Öffne den Tab **Servers** und suche deinen Server in der Liste. Per Rechtsklick kannst du ihn zu den **Favoriten** hinzufügen.

3. <b>Server beitreten</b><br>
   Wähle deinen Server aus und klicke auf **Connect**.

:::: info Hinweis
Der Standard-Port von Space Engineers ist `27016`. Auf deinem Server kann jedoch ein anderer Port vergeben sein — verwende immer den in der Verwaltung angezeigten Port.
::::

:::: tip Tipp
Damit dein Server im Tab **Servers** öffentlich sichtbar ist, muss er laufen und auf öffentlich (public) gestellt sein. Private Server erreichst du nur über die **Direktverbindung**.
::::
