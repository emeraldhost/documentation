# So richtest du automatische Backups auf deinem Space Engineers Server ein

Dein Server kann den Zustand der Welt in regelmäßigen Abständen automatisch sichern. So aktivierst oder deaktivierst du die automatischen Backups und legst das Intervall fest.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Automatische Backups aktivieren</b><br>
   Setze die Einstellung **Automatische Backups** auf `true`, um sie zu aktivieren, oder auf `false`, um sie zu deaktivieren.

4. <b>Intervall festlegen</b><br>
   Trage im Feld **Automatischer Backup Intervall** das gewünschte Intervall in Minuten ein, in dem der Server die Welt sichert (Standard: `5`).

5. <b>Server neu starten</b><br>
   Speichere die Einstellungen und starte deinen Server neu, damit die Änderungen übernommen werden.

:::: tip Tipp
Ein manuelles Backup kannst du jederzeit zusätzlich über die [Backup-Funktion](../backup-erstellen.md) in der Verwaltung erstellen.
::::
