# So änderst du die maximale Spieleranzahl auf deinem Space Engineers Server

Mit dieser Einstellung legst du fest, wie viele Spieler gleichzeitig auf deinem Server spielen können.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Maximale Spieler ändern</b><br>
   Trage im Feld **Maximale Spieler** die gewünschte Anzahl ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Höhere Spielerzahlen belasten den Server stärker. Sehr viele gleichzeitige Spieler können — besonders in Kombination mit großen Konstruktionen — die Leistung beeinträchtigen.
::::
