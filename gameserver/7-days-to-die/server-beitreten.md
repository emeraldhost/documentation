# So trittst du deinem 7 Days to Die Server bei

Du kannst deinem Server über das Spiel oder den Steam-Server-Browser beitreten.

## Über das Spiel

1. <b>Spiel starten</b><br>
   Starte 7 Days to Die.

2. <b>Einem Spiel beitreten</b><br>
   Klicke auf **Einem Spiel beitreten**.

3. <b>Mit IP verbinden</b><br>
   Klicke auf **Mit IP verbinden** und gib die IP-Adresse und den Port deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:26900
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Verbinden**.

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Navigiere zu **Anzeige → Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann auf **Server hinzufügen**.

4. <b>IP eingeben</b><br>
   Gib `IP:Port` deines Servers ein (z.B. `123.45.67.89:26900`).

5. <b>Verbinden</b><br>
   Wähle den Server aus und klicke auf **Verbinden**.

:::: info Hinweis
Der Standard-Port ist `26900`. Die IP-Adresse und den Port findest du in der Verwaltung deines Servers.
::::
