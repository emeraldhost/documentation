# So trittst du deinem 7 Days to Die Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht. Der Standard-Port ist `26900`.
::::

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann unten rechts auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den Port deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:26900
   ```
   ::::

5. <b>Verbinden</b><br>
   Wähle den Server aus und klicke auf **Verbinden**.

## Über den Servernamen suchen

1. <b>Spiel starten</b><br>
   Starte 7 Days to Die.

2. <b>Einem Spiel beitreten</b><br>
   Klicke auf **Einem Spiel beitreten**.

3. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers in der Server-Liste.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Verbinden**.

## Über Direct Connect

1. <b>Spiel starten</b><br>
   Starte 7 Days to Die.

2. <b>Einem Spiel beitreten</b><br>
   Klicke auf **Einem Spiel beitreten**.

3. <b>Mit IP verbinden</b><br>
   Klicke auf **Mit IP verbinden** und gib die IP-Adresse und den Port deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:26900
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Verbinden**.
