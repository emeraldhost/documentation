# So setzt du die Owner ID auf deinem RuneScape: Dragonwilds Server

Die Owner ID ist deine Epic Games ID und wird benötigt, damit dein Server starten kann.

:::: tip Tipp
Hier findest du eine Anleitung, wie du deine [Epic Games ID](../epicgamesid-herausfinden.md) herausfindest. Du kannst deine ID auch direkt im Spiel unter **Einstellungen** ganz unten bei **My Player ID** finden.
::::

## Owner ID eintragen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Owner ID eintragen</b><br>
   Trage deine Player ID im Feld **Owner ID** ein.

4. <b>Server starten</b><br>
   Speichere die Einstellung und starte deinen Server.

:::: danger Wichtig
Der Server wird **nicht starten**, solange keine Owner ID eingetragen ist.
::::
