# So trittst du deinem RuneScape: Dragonwilds Server bei

Du kannst deinem Server über den Welt-Namen im Spiel beitreten.

1. <b>Spiel starten</b><br>
   Starte RuneScape: Dragonwilds.

2. <b>Welten öffnen</b><br>
   Öffne den **Welten**-Bildschirm und wähle den Tab **Public**.

3. <b>Server suchen</b><br>
   Öffne die **Suche** und gib den exakten Welt-Namen deines Servers ein.

   :::: warning Achtung
   Der Welt-Name ist **groß-/kleinschreibungsempfindlich**. Er muss exakt übereinstimmen.
   ::::

4. <b>Server beitreten</b><br>
   Warte einige Sekunden, bis dein Server in der Liste erscheint, und tritt bei.

:::: info Hinweis
Den Welt-Namen findest du in den **Einstellungen** deines Servers in der Verwaltung.
::::

:::: tip Fehlerbehebung
Falls dein Server nicht gefunden wird:
- Stelle sicher, dass die [Owner ID](owner-id-setzen.md) hinterlegt ist
- Stelle sicher, dass der Welt-Name exakt übereinstimmt
- Prüfe, ob dein Spiel und der Server auf derselben Version sind
- Warte einige Minuten nach dem Serverstart, bevor du suchst
::::
