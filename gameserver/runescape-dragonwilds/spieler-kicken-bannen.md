# So kickst und bannst du Spieler auf deinem RuneScape Dragonwilds Server

Du kannst Spieler über das Ingame-Admin-Menü kicken und bannen. Dafür wird das Admin-Passwort benötigt.

## Spieler kicken oder bannen

1. <b>Pausemenü öffnen</b><br>
   Drücke die Pause-Taste, um das Menü zu öffnen.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu **Settings** im Pausemenü.

3. <b>Server Management öffnen</b><br>
   Wähle **Server Management** aus. Gib das Admin-Passwort ein, falls du dazu aufgefordert wirst.

4. <b>Spieler kicken oder bannen</b><br>
   Wähle den Spieler aus der Liste aus und klicke auf die entsprechende Option zum Kicken oder Bannen.

:::: info Hinweis
Du benötigst das Admin-Passwort, um auf die Server-Management-Optionen zugreifen zu können.
::::
