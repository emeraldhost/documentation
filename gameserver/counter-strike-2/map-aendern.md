# So änderst du die Map auf deinem Counter-Strike 2 Server

## Map über die Verwaltung ändern

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Map auswählen</b><br>
   Wähle im Dropdown **Map** die gewünschte Karte aus.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

## Map über die Konsole wechseln

Du kannst die Map auch live über die **Server-Konsole** in der Verwaltung wechseln:

```
changelevel de_dust2
```
