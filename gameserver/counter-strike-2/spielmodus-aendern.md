# So änderst du den Spielmodus auf deinem Counter-Strike 2 Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Gamemode auswählen</b><br>
   Wähle im Dropdown **Gamemode** den gewünschten Spielmodus aus.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

## Verfügbare Spielmodi

| Spielmodus | Beschreibung |
|------------|-------------|
| `competitive` | Klassisches 5v5 Ranked-Match |
| `wingman` | 2v2 auf kleineren Maps |
| `casual` | Lockeres Spiel mit größeren Teams |
| `deathmatch` | Freies Deathmatch ohne Runden |
| `custom` | Benutzerdefinierter Spielmodus |

:::: tip Tipp
Du kannst die Standard-Map über das Feld **Map** in den Einstellungen ändern.
::::
