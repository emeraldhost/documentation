# So kickst und bannst du Spieler auf deinem BeamMP Server

Du kannst Spieler über die Serverkonsole kicken und bannen.

## Spieler über die Serverkonsole kicken oder bannen

1. <b>Serverkonsole öffnen</b><br>
   Öffne die Konsole deines Servers über die Verwaltung.

2. <b>Spieler kicken oder bannen</b><br>
   Verwende einen der folgenden Befehle:

   ```
   kick <Name>
   ```

   ```
   ban <Name>
   ```

:::: info Hinweis
Ein gekickter Spieler kann dem Server sofort wieder beitreten. Ein gebannter Spieler wird dauerhaft vom Server ausgeschlossen.
::::

## Befehle im Überblick

| Befehl | Beschreibung |
|--------|-------------|
| `kick <Name>` | Spieler vom Server kicken |
| `ban <Name>` | Spieler dauerhaft bannen |
