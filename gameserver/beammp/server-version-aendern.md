# So änderst du die Server Version auf deinem BeamMP Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Version eintragen</b><br>
   Trage die gewünschte Version im Feld **BeamMP Server Version** ein.

   :::: tip Beispiel
   ```
   v3.1.0
   ```
   ::::

   Alle verfügbaren Versionen findest du auf [GitHub](https://github.com/BeamMP/BeamMP-Server/releases). Trage `latest` ein, um immer die neueste Version zu verwenden.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Ungültige Versionen werden automatisch auf die neueste Version zurückgesetzt.
::::
