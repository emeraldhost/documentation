# So änderst du die maximale Anzahl an Autos auf deinem BeamMP Server

Du kannst einstellen, wie viele Autos jeder Spieler gleichzeitig spawnen darf.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Max Autos einstellen</b><br>
   Trage die gewünschte Anzahl im Feld **Maximale Autos** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
