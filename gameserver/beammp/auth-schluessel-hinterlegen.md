# So hinterlegst du den Authentifizierungsschlüssel auf deinem BeamMP Server

Um deinen BeamMP Server starten zu können, benötigst du einen **Authentifizierungsschlüssel**.

1. <b>Schlüssel erstellen</b><br>
   Erstelle einen Schlüssel beim [BeamMP Keymaster](https://beammp.com/keymaster).

2. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

3. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

4. <b>Schlüssel eintragen</b><br>
   Trage den Schlüssel im Feld **Authentifizierungsschlüssel** ein.

5. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: danger Wichtig
Teile deinen Authentifizierungsschlüssel niemals mit anderen Personen.
::::
