# So aktivierst oder deaktivierst du Chat Logging auf deinem BeamMP Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Chat Logging einstellen</b><br>
   Trage im Feld **Chat Logging** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `true` | Chat Logging aktiviert |
   | `false` | Chat Logging deaktiviert |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
