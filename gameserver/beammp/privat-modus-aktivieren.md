description: Privat-Modus auf einem BeamMP Server aktivieren oder deaktivieren

# So aktivierst oder deaktivierst du den Privat-Modus auf deinem BeamMP Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Privat-Modus einstellen</b><br>
   Trage im Feld **Private** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `true` | Server ist privat und nicht in der Serverliste sichtbar |
   | `false` | Server ist öffentlich und in der Serverliste sichtbar (Standard) |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
