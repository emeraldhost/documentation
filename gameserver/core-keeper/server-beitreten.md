# So trittst du deinem Core Keeper Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die **Game ID** deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht oder in der Server-Konsole beim Start.
::::

## Über die Game ID

1. <b>Core Keeper starten</b><br>
   Starte Core Keeper.

2. <b>Charakter auswählen</b><br>
   Wähle deinen Charakter aus.

3. <b>Co-op beitreten</b><br>
   Klicke auf **Co-op** und wähle **Join Game**.

4. <b>Game ID eingeben</b><br>
   Gib die **Game ID** deines Servers in das Eingabefeld ein.

5. <b>Verbinden</b><br>
   Klicke auf **Join**, um dem Server beizutreten.
