# So änderst du den Welt Seed auf deinem Core Keeper Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Welt Seed eintragen</b><br>
   Trage den gewünschten Seed im Feld **Welt Seed** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Der Seed wird nur bei der Erstellung einer neuen Welt verwendet. Bei einer bestehenden Welt hat eine Änderung keine Auswirkung.
::::
