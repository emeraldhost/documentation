# So änderst du das Content Bundle auf deinem Core Keeper Server

Mit Content Bundles kannst du zusätzliche Inhalte auf deinem Server aktivieren.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Content Bundle eintragen</b><br>
   Trage im Feld **Content Bundle** den gewünschten Bundle-Namen ein. Für mehrere Bundles trenne die Namen mit Kommas ohne Leerzeichen.

   :::: tip Beispiel
   ```
   GiantCicadaBossDungeon,NatureBiomeCicadas,GuaranteedOases
   ```
   ::::

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Wenn das Feld leer gelassen wird, lädt der Server das Standard-Bundle.
::::
