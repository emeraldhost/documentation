# So änderst du den Welt Index auf deinem Core Keeper Server

Der Welt Index bestimmt, welche Welt geladen wird. Er muss mit der Zahl deiner Welt-Datei übereinstimmen (z.B. `0` für `0.world.gzip`).

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Welt Index eintragen</b><br>
   Trage die gewünschte Zahl im Feld **Welt Index** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: tip Beispiel
Wenn deine Welt-Datei `3.world.gzip` heißt, muss der Welt Index `3` sein.
::::
