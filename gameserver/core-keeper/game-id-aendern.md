# So änderst du die Game ID auf deinem Core Keeper Server

Die Game ID ist die eindeutige Kennung deines Servers. Spieler benötigen diese, um dem Server beizutreten.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Game ID eintragen</b><br>
   Trage die gewünschte ID im Feld **Game ID** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Die Game ID muss mindestens **28 Zeichen** lang und alphanumerisch sein. Folgende Zeichen sind **nicht** erlaubt: `Y`, `y`, `x`, `0`, `O`. Wenn das Feld leer oder ungültig ist, wird beim Start automatisch eine neue ID generiert.
::::
