# So trittst du deinem ARK Survival Evolved Server bei

Wir empfehlen dir, deinen Server über den **Steam-Server-Browser** hinzuzufügen, um ihn ingame direkt zu finden.

## Verbindungsdaten finden

:::: warning Wichtig
Für die Verbindung benötigst du die **IP-Adresse** und den **Query Port** deines Servers. Den Query Port findest du in den **Einstellungen** deines Servers in der Verwaltung. Verwende **nicht** den Gameport, sondern den Query Port!
::::

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann unten rechts auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den **Query Port** deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>ARK starten</b><br>
   Starte ARK Survival Evolved.

6. <b>Server suchen</b><br>
   Wähle im Hauptmenü **Join ARK** und filtere nach **Favoriten** — dein Server sollte nun in der Liste angezeigt werden.

7. <b>Server beitreten</b><br>
   Wähle deinen Server aus und klicke auf **Join**.

## Über den Servernamen suchen

1. <b>ARK starten</b><br>
   Starte ARK Survival Evolved.

2. <b>Join ARK öffnen</b><br>
   Wähle im Hauptmenü **Join ARK**.

3. <b>Filter setzen</b><br>
   Stelle den Session-Filter auf **Unofficial**.

4. <b>Servername eingeben</b><br>
   Gib den Namen deines Servers in das Suchfeld **Name Filter** ein.

5. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Join**.

## Über die Ingame-Konsole

1. <b>ARK starten</b><br>
   Starte ARK Survival Evolved und wähle im Hauptmenü **Join ARK**.

2. <b>Konsole öffnen</b><br>
   Drücke `Tab`, um die Konsole zu öffnen.

3. <b>Verbinden</b><br>
   Gib folgenden Befehl ein:

   ```
   open IP:Port
   ```

   :::: tip Beispiel
   ```
   open 123.45.67.89:27015
   ```
   ::::

:::: info Hinweis
Für die Ingame-Konsole musst du den **Gameport** (nicht den Query Port) verwenden.
::::
