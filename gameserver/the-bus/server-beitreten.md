# So trittst du einem The Bus Server bei

Um einem The Bus Server beizutreten, benötigst du die **IP-Adresse und den Port** deines Servers. Diese findest du in der Verwaltung deines Servers unter "Übersicht".

:::: info Server-GUID
Beim Start des Dedicated Servers wird eine **GUID** in der Serverkonsole angezeigt. Diese kannst du alternativ zur IP-Adresse verwenden, um dich zu verbinden.
::::

## So stellst du die Verbindung her

1. <b>The Bus starten</b><br>
   Starte The Bus und öffne das `Multiplayer`-Menü.

2. <b>Server hinzufügen</b><br>
   Drücke die `Alt`-Taste, um einen neuen Server hinzuzufügen.

3. <b>Server-Adresse eingeben</b><br>
   Gib die Server-GUID oder die IP-Adresse mit Port ein (z.B. `123.45.67.89:7777`).

4. <b>Verbinden</b><br>
   Verbinde dich mit dem Server.
