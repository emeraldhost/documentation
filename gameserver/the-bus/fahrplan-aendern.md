# So änderst du den Fahrplan auf einem The Bus Server

Du kannst den Fahrplan (Operating Plan) über das **Admin-Menü** oder per **Befehl** ändern.

## Fahrplan über das Admin-Menü ändern

1. <b>Admin-Menü öffnen</b><br>
   Öffne das Pausenmenü und wähle das **Admin-Menü**. Du benötigst Owner- oder Admin-Rechte.

2. <b>Fahrplan auswählen</b><br>
   Wähle den gewünschten Fahrplan aus den verfügbaren Optionen.

## Fahrplan per Befehl ändern

Gib folgenden Befehl im Ingame-Chat ein:

```
/operatingPlan <plan>
```

:::: tip Tipp
Dieser Befehl erfordert Owner- oder Admin-Rechte.
::::
