# So änderst du die Map auf einem The Bus Server

Du kannst die aktive Karte über das **Admin-Menü** oder per **Befehl** ändern.

## Map über das Admin-Menü ändern

1. <b>Admin-Menü öffnen</b><br>
   Öffne das Pausenmenü und wähle das **Admin-Menü**. Du benötigst Owner- oder Admin-Rechte.

2. <b>Map auswählen</b><br>
   Wähle die gewünschte Karte aus den verfügbaren Optionen.

## Map per Befehl ändern

Gib folgenden Befehl im Ingame-Chat ein:

```
/map <map>
```

## Verfügbare Maps anzeigen

Um alle verfügbaren Maps aufzulisten:

```
/mapList
```

:::: tip Tipp
Diese Befehle erfordern Owner- oder Admin-Rechte.
::::
