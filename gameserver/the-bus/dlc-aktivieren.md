# So aktivierst du DLCs auf einem The Bus Server

Du kannst DLCs auf deinem Server per Befehl aktivieren oder deaktivieren.

## So aktivierst oder deaktivierst du ein DLC

Gib folgenden Befehl im Ingame-Chat ein:

```
/dlc
```

:::: tip Tipp
Dieser Befehl erfordert Owner- oder Admin-Rechte.
::::
