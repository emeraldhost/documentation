# So änderst du Tageszeit und Datum auf einem The Bus Server

Du kannst die Tageszeit und das Datum auf deinem Server per Befehl ändern.

## So änderst du die Tageszeit

Gib folgenden Befehl in die Konsole oder den Ingame-Chat ein:

```
/time <stunde>
```

**Beispiele:**

```
/time 6
/time 12
/time 20
```

| Wert | Beschreibung |
|------|-------------|
| `6` | Morgen |
| `12` | Mittag |
| `18` | Abend |
| `0` | Mitternacht |

## So änderst du das Datum

```
/date <datum>
```

:::: tip Tipp
Diese Befehle können über die Konsole in der Verwaltung oder direkt im Ingame-Chat eingegeben werden.
::::
