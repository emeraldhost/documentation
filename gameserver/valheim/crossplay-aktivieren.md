# So aktivierst oder deaktivierst du Crossplay auf deinem Valheim Server

Mit Crossplay können Spieler von verschiedenen Plattformen (Steam, Xbox) gemeinsam spielen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Crossplay einstellen</b><br>
   Trage im Feld **Enable Crossplay** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `1` | Crossplay aktiviert |
   | `0` | Crossplay deaktiviert |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
