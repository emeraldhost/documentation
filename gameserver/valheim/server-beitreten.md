# So trittst du deinem Valheim Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über den Servernamen suchen

1. <b>Valheim starten</b><br>
   Starte Valheim und wähle deinen Charakter aus.

2. <b>Beitreten öffnen</b><br>
   Klicke auf **Beitreten** und wähle den Tab **Community**.

3. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers in der Server-Liste.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Verbinden**.

## Über Direct Connect

1. <b>Valheim starten</b><br>
   Starte Valheim und wähle deinen Charakter aus.

2. <b>Beitreten öffnen</b><br>
   Klicke auf **Beitreten**.

3. <b>IP eingeben</b><br>
   Klicke unten auf **IP beitreten** und gib die IP-Adresse sowie den Port deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:2456
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Verbinden**, um dem Server beizutreten.
