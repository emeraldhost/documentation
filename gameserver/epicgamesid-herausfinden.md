# Epic Games ID herausfinden

## Epic Games ID über den Webbrowser herausfinden

1. <b>Website öffnen</b><br>
   Gehe auf die [Epic Games Website](https://www.epicgames.com/site/home).

2. <b>Anmelden</b><br>
   Klicke auf "Anmelden" und melde Dich mit Deinem Epic Games-Konto an.

3. <b>ID finden</b><br>
   Klicke auf "Konto", dann findest Du Deine Epic Games ID unter "Kontoinformationen".

## Epic Games ID über die Launcher-Datei herausfinden

1. <b>Ordner öffnen</b><br>
   Drücke die `Windows-Taste + R`, gib dort `AppData` ein und drücke die `Enter-Taste`.

2. <b>Zum Ordner navigieren</b><br>
   Navigiere zu folgendem Ordner:

    ```
    Local/EpicGamesLauncher/Saved/Data
    ```

3. <b>ID ablesen</b><br>
   Du solltest nun zwei `.dat`-Dateien sehen. Die Epic Games ID entspricht dem Dateinamen.
