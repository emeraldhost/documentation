# So trittst du deinem Palworld Server bei

Du kannst deinem Server über die Direktverbindung im Spiel beitreten.

:::: info Hinweis
Die Serveradresse inklusive Port findest du in der Verwaltung deines Gameservers unter der Übersicht.
::::

1. <b>Palworld starten</b><br>
   Starte das Spiel Palworld.

2. <b>Mehrspieler-Modus auswählen</b><br>
   Wähle im Hauptmenü die Option **Mehrspieler starten**.
   <img src="../../assets/gameserver/palworld/server-beitreten/schritt1.jpg" alt="Mehrspieler-Modus auswählen">

3. <b>Serveradresse eingeben</b><br>
   Gib deine Serveradresse im Eingabefeld für die Direktverbindung ein.
   <img src="../../assets/gameserver/palworld/server-beitreten/schritt2.png" alt="Serveradresse eingeben">

4. <b>Verbindung herstellen</b><br>
   Klicke auf **Verbinden**, um dich mit dem Server zu verbinden.
   <img src="../../assets/gameserver/palworld/server-beitreten/schritt3.png" alt="Verbindung herstellen">
