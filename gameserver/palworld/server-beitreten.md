# So trittst du deinem Palworld Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die Serveradresse inklusive Port findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Palworld starten</b><br>
   Starte Palworld.

2. <b>Mehrspieler-Modus auswählen</b><br>
   Wähle im Hauptmenü **Mehrspieler starten**.

3. <b>Serveradresse eingeben</b><br>
   Gib deine Serveradresse im Eingabefeld für die Direktverbindung ein.

   :::: tip Beispiel
   ```
   123.45.67.89:8211
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Verbinden**, um dem Server beizutreten.

## Über den Servernamen suchen

1. <b>Palworld starten</b><br>
   Starte Palworld und wähle im Hauptmenü **Mehrspieler starten**.

2. <b>Community-Server öffnen</b><br>
   Öffne die Liste der Community-Server.

3. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Verbinden**.
