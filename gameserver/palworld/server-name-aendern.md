# So änderst du den Server Namen deines Palworld Servers

Du kannst den Anzeigenamen deines Servers ändern, der in der Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Name ändern</b><br>
   Trage den gewünschten Namen ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: warning Achtung
Setze den Server Namen nicht direkt in der `PalWorldSettings.ini`, da er beim nächsten Serverstart vom Webinterface überschrieben wird.
::::
