# So sendest du eine Broadcast Nachricht auf deinem Palworld Server

Mit dem `/Broadcast`-Befehl kannst du eine Nachricht an alle Spieler auf dem Server senden.

:::: info Hinweis
Du benötigst Admin-Rechte, um diesen Befehl nutzen zu können. Siehe [Admin hinzufügen](admin-hinzufuegen.md).
::::

## Broadcast senden

1. <b>Chat öffnen</b><br>
   Drücke `Enter`, um den Chat zu öffnen.

2. <b>Befehl eingeben</b><br>
   Gib folgenden Befehl ein:

   ```
   /Broadcast <Nachricht>
   ```

   Ersetze `<Nachricht>` durch den gewünschten Text.

:::: tip Beispiel
```
/Broadcast Server wird in 10 Minuten neugestartet!
```
::::
