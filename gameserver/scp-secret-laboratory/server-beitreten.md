# So trittst du deinem SCP: Secret Laboratory Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>SCP: Secret Laboratory starten</b><br>
   Starte SCP: Secret Laboratory über Steam.

2. <b>Servers öffnen</b><br>
   Klicke im Hauptmenü auf **Servers**.

3. <b>Direct Connect</b><br>
   Wähle die Option **Direct Connect** und gib die IP-Adresse sowie den Port deines Servers ein:

   :::: tip Beispiel
   ```
   123.45.67.89:7777
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Connect**, um dem Server beizutreten.

## Über die Favoriten

Alternativ kannst du deinen Server als Steam-Favorit hinzufügen, damit er dauerhaft in deiner Liste erscheint.

1. <b>Steam öffnen</b><br>
   Öffne Steam und klicke oben auf **Anzeigen → Server**.

2. <b>Favoriten öffnen</b><br>
   Wechsle zum Tab **Favoriten** und klicke unten auf **Server hinzufügen**.

3. <b>Server-Adresse eingeben</b><br>
   Trage die IP-Adresse und den Port deines Servers ein und klicke auf **Hinzufügen zu Favoriten**.

4. <b>Beitreten</b><br>
   Wähle deinen Server aus den Favoriten aus und klicke auf **Verbinden**.
