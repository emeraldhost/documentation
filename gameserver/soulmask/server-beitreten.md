# So trittst du deinem Soulmask Server bei

## Verbindungsdaten finden

:::: warning Wichtig
Für die Verbindung benötigst du die **IP-Adresse** und den **Query Port** deines Servers. Den Query Port findest du in den **Einstellungen** deines Servers in der Verwaltung. Verwende **nicht** den Gameport, sondern den Query Port!
::::

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann unten rechts auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den **Query Port** deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>Soulmask starten</b><br>
   Starte Soulmask.

6. <b>Server beitreten</b><br>
   Dein Server sollte nun in der Favoritenliste angezeigt werden. Wähle ihn aus und klicke auf **Verbinden**.

## Über den Servernamen suchen

1. <b>Soulmask starten</b><br>
   Starte Soulmask.

2. <b>Online-Spiel wählen</b><br>
   Wähle im Hauptmenü **Online-Spiel**.

3. <b>Private Server wählen</b><br>
   Wähle den Tab **Private Server**.

4. <b>Server suchen</b><br>
   Suche oben rechts im Suchfeld nach dem Namen deines Servers.

5. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und tritt bei.

## Über Direct Connect

1. <b>Soulmask starten</b><br>
   Starte Soulmask.

2. <b>Online-Spiel wählen</b><br>
   Wähle im Hauptmenü **Online-Spiel**.

3. <b>Direkt verbinden</b><br>
   Klicke unten auf **Direkt mit Server verbinden**.

4. <b>Server-Daten eingeben</b><br>
   Gib die **IP-Adresse**, den **Port** und optional das **Server Passwort** ein.

5. <b>Verbinden</b><br>
   Klicke auf **Verbinden**, um dem Server beizutreten.
