# So fügst du einen Admin auf deinem Soulmask Server hinzu

Du kannst dir und anderen Spielern Admin-Rechte im Spiel vergeben.

## Voraussetzung

Bevor du dich als Admin anmelden kannst, muss ein Admin Passwort auf deinem Server eingerichtet sein. Folge dazu der Anleitung [Admin Passwort setzen](admin-passwort-setzen.md).

## Als Admin anmelden

1. <b>Server beitreten</b><br>
   Tritt deinem Server bei.

2. <b>Konsole öffnen</b><br>
   Drücke `~`, um die Konsole zu öffnen.

3. <b>Admin-Login</b><br>
   Gib folgenden Befehl ein:

   ```
   gm key DeinAdminPasswort
   ```

:::: info Hinweis
Die Admin-Anmeldung gilt nur für die aktuelle Sitzung. Nach einem Neuverbinden musst du den Befehl erneut eingeben.
::::
