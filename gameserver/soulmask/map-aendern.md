# So änderst du die Map auf deinem Soulmask Server

Du kannst zwischen den verfügbaren Maps wechseln.

:::: warning Achtung
Für die Shifting Sands Map benötigst du das **Shifting Sands DLC**. Alle Spieler müssen das DLC ebenfalls besitzen. Beim Map-Wechsel werden Spieler aufgefordert, einen neuen Charakter zu erstellen.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Map auswählen</b><br>
   Ändere die Map-Einstellung auf den gewünschten Wert:

   | Wert | Map |
   |------|-----|
   | `Level01_Main` | Cloud Mist Forest (Standard) |
   | `DLC_Level01_Main` | Shifting Sands (DLC) |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: tip Tipp
Wenn du beide Maps gleichzeitig betreiben und Spielern den Wechsel zwischen den Maps ermöglichen möchtest, benötigst du einen Server-Cluster mit zwei Servern. Siehe [Shifting Sands einrichten](shifting-sands-einrichten.md).
::::
