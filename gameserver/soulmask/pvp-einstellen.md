# So stellst du PvP und PvE auf deinem Soulmask Server ein

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Spielmodus eintragen</b><br>
   Trage im Feld **Spielmodus** den gewünschten Modus ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `pve` | Spieler können sich nicht gegenseitig angreifen |
   | `pvp` | Spieler können sich jederzeit angreifen |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
