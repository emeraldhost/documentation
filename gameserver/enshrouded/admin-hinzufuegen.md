# So fügst du einen Admin auf deinem Enshrouded Server hinzu

In Enshrouded wird der erste Spieler, der dem Server mit dem Admin-Passwort beitritt, zum Host. Du kannst das Admin-Passwort über die Verwaltung festlegen.

## Admin-Passwort festlegen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung und wähle deinen Enshrouded Server aus.

2. <b>Einstellungen aufrufen</b><br>
   Navigiere zu den Einstellungen deines Servers.

3. <b>Admin-Passwort setzen</b><br>
   Trage unter `Admin Passwort` dein gewünschtes Passwort ein.

4. <b>Server neustarten</b><br>
   Speichere die Einstellungen und starte den Server neu.

## Als Admin beitreten

1. <b>Server beitreten</b><br>
   Verbinde dich mit deinem Enshrouded Server.

2. <b>Admin-Passwort eingeben</b><br>
   Gib beim Beitritt das zuvor festgelegte Admin-Passwort ein. Der erste Spieler, der das korrekte Passwort eingibt, wird zum Host.

:::: info Hinweis
Der Host hat die vollen Verwaltungsrechte auf dem Server. Teile das Admin-Passwort nur mit vertrauenswürdigen Spielern.
::::

:::: tip Tipp
Du kannst das Admin-Passwort auch direkt in der Datei `enshrouded_server.json` auf dem Server ändern. Verbinde dich dazu per [SFTP](../sftp-verbindung-herstellen.md) mit deinem Server.
::::
