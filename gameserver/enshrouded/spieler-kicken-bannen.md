# So kickst und bannst du Spieler auf deinem Enshrouded Server

In Enshrouded kannst du Spieler über die Ingame-Spielerliste kicken. Ein dauerhaftes Ban-System ist derzeit nicht verfügbar.

## Spieler über die Spielerliste kicken

1. <b>Server beitreten</b><br>
   Tritt deinem Enshrouded Server als Host oder Admin bei.

2. <b>Spielerliste öffnen</b><br>
   Öffne die Spielerliste im Spiel.

3. <b>Spieler kicken</b><br>
   Wähle den Spieler aus und klicke auf **Kick**, um ihn vom Server zu entfernen.

:::: info Hinweis
Enshrouded bietet derzeit kein dauerhaftes Ban-System. Ein gekickter Spieler kann dem Server erneut beitreten.
::::

## Zugang über Server-Passwort einschränken

Wenn du bestimmte Spieler dauerhaft vom Server ausschließen möchtest, kannst du ein Server-Passwort setzen:

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Passwort setzen</b><br>
   Trage ein Passwort im Feld **Server Passwort** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: tip Tipp
Teile das Passwort nur mit Spielern, denen du vertraust. Ändere das Passwort, wenn du einen Spieler ausschließen möchtest.
::::
