# So änderst du den Welt Namen deines FOUNDRY Servers

Der Welt Name legt fest, welcher Welt-Ordner auf dem Server geladen wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Welt Name ändern</b><br>
   Trage im Feld **Welt Name** den gewünschten Namen ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
