# So trittst du deinem FOUNDRY Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über IP Direct Connect

1. <b>FOUNDRY starten</b><br>
   Starte FOUNDRY über Steam.

2. <b>Multiplayer öffnen</b><br>
   Klicke im Hauptmenü auf **Multiplayer**.

3. <b>IP Direct Connect</b><br>
   Wähle die Option **IP Direct Connect** und gib die IP-Adresse sowie den Port deines Servers ein:

   :::: tip Beispiel
   ```
   123.45.67.89:3724
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Connect**, um dem Server beizutreten.

## Über den Steam Server-Browser

Öffentliche FOUNDRY-Server kannst du auch über den Steam Server-Browser finden.

1. <b>Steam Server-Browser öffnen</b><br>
   Öffne Steam und gehe zu **Anzeigen → Server**.

2. <b>Favoriten öffnen</b><br>
   Wechsle zum Tab **Favoriten** und klicke unten auf **Server hinzufügen**.

3. <b>Server-Adresse eingeben</b><br>
   Trage die IP-Adresse und den Port deines Servers ein und bestätige.

4. <b>Beitreten</b><br>
   Wähle deinen Server aus den Favoriten aus und klicke auf **Verbinden**.
