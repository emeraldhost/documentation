# So aktivierst du die Serverliste deines FOUNDRY Servers

Wenn die Serverliste aktiviert ist, wird dein Server im Steam Server-Browser gelistet und ist dort öffentlich auffindbar.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Serverliste aktivieren</b><br>
   Aktiviere die Option **Serverliste** und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Info
Ist die Serverliste aktiviert, wird dein Server im Steam Server-Browser angezeigt und kann dort von anderen Spielern gefunden werden.
::::
