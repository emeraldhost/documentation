# So hältst du deinen FOUNDRY Server automatisch an, wenn er leer ist

Mit dieser Einstellung pausiert dein Server, sobald kein Spieler mehr verbunden ist. Das spart CPU und RAM, bis wieder jemand beitritt.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Option aktivieren</b><br>
   Aktiviere die Option **Server anhalten, wenn er leer ist** und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Info
Hält der Server an, wenn niemand verbunden ist. Dadurch werden CPU und RAM geschont, bis wieder ein Spieler beitritt.
::::
