# So setzt du die Island ID auf deinem Windrose Server

Die Island ID verknüpft deinen Server mit einer bestimmten Welt. Sie ist in der Datei `WorldDescription.json` zu finden. Wenn keine Island ID festgelegt ist, wird automatisch der aktuellste Spielstand aus dem Verzeichnis `/R5/Saved/` verwendet.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Island ID setzen</b><br>
   Trage die gewünschte Island ID im Feld **Island ID** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
