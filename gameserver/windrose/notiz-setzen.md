# So setzt du eine Notiz auf deinem Windrose Server

Du kannst eine Notiz mit zusätzlichen Informationen zu deinem Server hinterlegen. Diese hilft in seltenen Fällen, deinen Server von anderen zu unterscheiden, wenn sich Einladungscodes zwischen zwei Servern ähneln.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Notiz setzen</b><br>
   Trage die gewünschten Informationen im Feld **Notiz** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
