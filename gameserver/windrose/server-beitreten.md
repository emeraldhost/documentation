# So trittst du deinem Windrose Server bei

Um deinem Server beizutreten, benötigst du den [Invite Code](invite-code-setzen.md) und – falls gesetzt – das [Server Passwort](server-passwort-setzen.md).

1. <b>Windrose starten</b><br>
   Starte Windrose über Steam.

2. <b>Spielen auswählen</b><br>
   Klicke im Hauptmenü auf **Spielen**.

3. <b>Mit Server verbinden</b><br>
   Wähle die Option **Mit Server verbinden** aus.

4. <b>Einladungscode eingeben</b><br>
   Gib den Invite Code deines Servers ein und klicke auf **Suchen**.

5. <b>Server auswählen</b><br>
   Der gefundene Server wird rechts angezeigt. Klicke auf den Eintrag, um dich zu verbinden.

6. <b>Passwort eingeben</b><br>
   Falls der Server ein Passwort hat, gib dieses ein, um beizutreten.

:::: info Hinweis
Deinen Invite Code findest du in der Verwaltung unter **Einstellungen** oder in der Datei `/R5/ServerDescription.json`. Der Code unterscheidet zwischen Groß- und Kleinschreibung – achte beim Eingeben auf die korrekte Schreibweise.
::::
