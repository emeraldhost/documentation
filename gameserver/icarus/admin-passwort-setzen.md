# So setzt du ein Admin Passwort auf deinem Icarus Server

Das Admin Passwort wird benötigt, um sich im Spiel als Administrator anzumelden.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Admin Passwort setzen</b><br>
   Trage das gewünschte Admin Passwort im Feld **Admin Passwort** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: tip Tipp
Teile das Admin Passwort nur mit vertrauenswürdigen Spielern. Jeder mit dem Passwort kann sich Administrator-Rechte vergeben.
::::
