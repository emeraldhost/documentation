# So trittst du deinem Icarus Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Icarus starten</b><br>
   Starte Icarus über Steam.

2. <b>Play auswählen</b><br>
   Klicke im Hauptmenü auf **Play**.

3. <b>Charakter wählen</b><br>
   Wähle deinen Charakter aus.

4. <b>Join auswählen</b><br>
   Klicke auf **Join** und wähle den Bereich **Dedicated Servers**.

5. <b>Direct Connect</b><br>
   Wähle die Option **Direct Connect** und gib die IP-Adresse sowie den Port deines Servers ein:

   :::: tip Beispiel
   ```
   123.45.67.89:17777
   ```
   ::::

6. <b>Verbinden</b><br>
   Klicke auf **Connect**, um dem Server beizutreten.

## Über die Serverliste

Alternativ kannst du deinen Server auch über die Serverliste finden.

1. <b>Dedicated Servers öffnen</b><br>
   Wähle im Join-Menü oben rechts **Dedicated Servers (Beta)**.

2. <b>Server suchen</b><br>
   Suche nach dem Namen, den du deinem Server gegeben hast.

3. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Join**.

:::: warning Achtung
Deine Icarus-Client-Version muss mit der Server-Version übereinstimmen. Prüfe Steam auf Updates, falls der Beitritt fehlschlägt.
::::
