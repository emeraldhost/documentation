# So trittst du deinem Project Zomboid Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Project Zomboid starten</b><br>
   Starte Project Zomboid.

2. <b>Multiplayer öffnen</b><br>
   Klicke im Hauptmenü auf **Mehrspieler**.

3. <b>Server beitreten</b><br>
   Wähle **Server beitreten**.

4. <b>Server-Daten eingeben</b><br>
   Gib einen Namen, die IP-Adresse und den Port deines Servers ein.

   :::: tip Beispiel
   ```
   IP: 123.45.67.89
   Port: 16261
   ```
   ::::

5. <b>Account-Daten eingeben</b><br>
   Gib deinen Benutzernamen und ggf. das Server Passwort ein.

6. <b>Verbinden</b><br>
   Klicke auf **Server beitreten**, um dem Server beizutreten.
