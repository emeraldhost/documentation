# So setzt du ein Server Passwort auf deinem Project Zomboid Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Passwort setzen</b><br>
   Trage das gewünschte Passwort im Feld **Server Passwort** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: tip Tipp
Um das Passwort zu entfernen und den Server wieder öffentlich zu machen, lösche den Wert im Feld und speichere.
::::
