# So aktivierst oder deaktivierst du die Serverliste auf deinem Project Zomboid Server

Du kannst einstellen, ob dein Server in der öffentlichen Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Serverliste einstellen</b><br>
   Trage im Feld **Serverliste** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `true` | Server wird in der öffentlichen Serverliste angezeigt |
   | `false` | Server wird nicht in der Serverliste angezeigt (nur per Direct Connect erreichbar) |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
