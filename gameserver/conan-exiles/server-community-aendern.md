# So änderst du die Server Community auf deinem Conan Exiles Server

Die Community wirkt sich darauf aus, wie dein Server in der Serverliste gefiltert wird. Wähle eine Community, damit Spieler einen Server finden, der ihrem Spielstil entspricht.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Community auswählen</b><br>
   Trage im Feld **Server Community** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `Purist` | Standard-Spielerlebnis (Standard) |
   | `Relaxed` | Entspanntes Spielerlebnis |
   | `HardCore` | Herausforderndes Spielerlebnis |
   | `RolePlaying` | Rollenspiel-fokussiert |
   | `Experimental` | Experimentelle Einstellungen |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
