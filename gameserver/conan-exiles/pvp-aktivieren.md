# So aktivierst oder deaktivierst du PvP auf deinem Conan Exiles Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>PvP einstellen</b><br>
   Trage im Feld **PvP** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `true` | PvP aktiviert — Spieler können sich gegenseitig angreifen |
   | `false` | PvP deaktiviert — Spieler können sich nicht angreifen (PvE) |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
