# So änderst du die Server Region auf deinem Conan Exiles Server

Die Serverregion wirkt sich darauf aus, wie dein Server in der Serverliste gefiltert wird. Wähle eine Region, um Spielern zu helfen, einen Server in ihrer Nähe zu finden.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Region auswählen</b><br>
   Trage im Feld **Server Region** den gewünschten Wert ein:

   | Wert | Region |
   |------|--------|
   | `0` | Europe (Standard) |
   | `1` | North America |
   | `2` | Asia |
   | `3` | Australia |
   | `4` | South America |
   | `5` | Japan |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
