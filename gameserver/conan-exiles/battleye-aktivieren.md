# So aktivierst oder deaktivierst du BattlEye auf deinem Conan Exiles Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>BattlEye einstellen</b><br>
   Trage im Feld **BattlEye** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `1` | BattlEye aktiviert |
   | `0` | BattlEye deaktiviert |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Wenn BattlEye aktiviert ist, müssen alle Spieler BattlEye installiert haben, um dem Server beitreten zu können.
::::
