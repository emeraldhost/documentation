# So fügst du einen Admin auf deinem Conan Exiles Server hinzu

## Admin Passwort festlegen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Admin Passwort setzen</b><br>
   Trage das gewünschte Passwort im Feld **Admin Passwort** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

## Als Admin anmelden

1. <b>Server beitreten</b><br>
   Tritt deinem Server bei.

2. <b>Einstellungen öffnen</b><br>
   Öffne im Spiel die **Einstellungen** und navigiere zu **Server-Einstellungen**.

3. <b>Admin Passwort eingeben</b><br>
   Trage dein Admin Passwort in das entsprechende Feld ein.

4. <b>Admin aktivieren</b><br>
   Klicke auf **Make Me Admin**, um dich als Admin anzumelden.

:::: info Hinweis
Die Admin-Anmeldung gilt nur für die aktuelle Sitzung. Nach einem Neuverbinden musst du dich erneut als Admin anmelden.
::::

:::: tip Tipp
Teile das Admin Passwort nur mit vertrauenswürdigen Spielern. Jeder mit dem Passwort kann sich Admin-Rechte vergeben.
::::
