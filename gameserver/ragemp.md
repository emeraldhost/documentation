# RAGE:MP

## Was ist RAGE:MP?

RAGE:MP (RAGE Multiplayer) ist eine alternative Multiplayer-Modifikation für Grand Theft Auto V. Es ermöglicht Spielern, eigene Server mit individuellen Spielmodi, Fahrzeugen und Maps zu erstellen und zu hosten.

Entwickler können mit C#, JavaScript und C++ eigene Gamemodes programmieren und so einzigartige Spielerlebnisse schaffen — von Roleplay-Servern über Racing bis hin zu komplett eigenen Spielmodi. RAGE:MP läuft vollständig eigenständig und ist nicht von Drittanbieter-Mods abhängig.

RAGE:MP unterstützt eine unbegrenzte Spieleranzahl und bietet eine leistungsstarke API für serverseitige und clientseitige Skripte. Das Spiel ist auf PC verfügbar.
