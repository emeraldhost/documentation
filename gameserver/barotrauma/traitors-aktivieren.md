# So aktivierst du Traitors auf deinem Barotrauma Server

Du kannst den Traitor-Modus direkt im Spiel als Admin über die Spielmodus-Einstellungen aktivieren.

1. <b>Spiel als Admin betreten</b><br>
   Tritt deinem Server bei und stelle sicher, dass du Admin-Rechte hast.

2. <b>Spielmodus-Einstellungen öffnen</b><br>
   Öffne die **Game Mode Settings** im Server-Menü.

3. <b>Traitor Settings öffnen</b><br>
   Navigiere zu den **Traitor Settings**.

4. <b>Traitor-Wahrscheinlichkeit einstellen</b><br>
   Setze die **Traitor Probability** auf einen Wert größer als **0**, um Traitors zu aktivieren.

5. <b>Maximales Gefahrenlevel einstellen</b><br>
   Lege das **Max Danger Level** fest, um die maximale Schwierigkeit der Traitor-Missionen zu bestimmen.

6. <b>Mindestanzahl an Spielern festlegen</b><br>
   Stelle die Mindestanzahl an Spielern ein, ab der ein Traitor ausgewählt werden kann.

7. <b>Runde starten</b><br>
   Übernimm die Einstellungen, indem du eine neue Runde startest.

:::: info Hinweis
Die Traitor-Einstellungen gelten ab der nächsten Runde. Je höher die Traitor Probability, desto wahrscheinlicher wird ein Spieler zum Verräter.
::::
