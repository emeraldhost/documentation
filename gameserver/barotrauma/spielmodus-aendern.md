# So änderst du den Spielmodus auf deinem Barotrauma Server

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Spielmodus auswählen</b><br>
   Trage im Feld **Gamemode** den gewünschten Wert ein:

   | Wert | Beschreibung |
   |------|-------------|
   | `Sandbox` | Freies Spielen ohne Einschränkungen |
   | `Mission` | Einzelne Missionen spielen |
   | `Campaign` | Kampagnen-Modus mit Fortschritt |

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
