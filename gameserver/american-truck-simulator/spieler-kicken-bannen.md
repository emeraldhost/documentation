# So kickst und bannst du Spieler auf deinem American Truck Simulator Server

Du kannst Spieler über das Ingame-Konvoi-Menü aus deiner Sitzung entfernen.

## Spieler über das Konvoi-Menü kicken

1. <b>Server beitreten</b><br>
   Tritt deinem American Truck Simulator Server bei.

2. <b>Konvoi-Menü öffnen</b><br>
   Öffne das Konvoi-Menü im Spiel.

3. <b>Spieler auswählen</b><br>
   Wähle den Spieler aus, den du entfernen möchtest.

4. <b>Spieler kicken</b><br>
   Klicke auf **Kick**, um den Spieler aus der Sitzung zu entfernen.

:::: info Hinweis
Ein gekickter Spieler kann der Sitzung erneut beitreten. Es gibt keine dauerhafte Ban-Funktion im Konvoi-Modus.
::::
