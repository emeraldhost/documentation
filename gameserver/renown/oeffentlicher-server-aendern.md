# So aktivierst du den Öffentlichen Server auf deinem Renown Server

Mit dieser Einstellung legst du fest, ob dein Server in der öffentlichen Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Öffentlicher Server anpassen</b><br>
   Setze das Feld **Öffentlicher Server** auf den gewünschten Wert:
   - `True` — Öffentlicher Server ist aktiviert
   - `False` — Öffentlicher Server ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: warning Wichtig
Wenn **Öffentlicher Server** auf `True` gesetzt ist, muss [Private Sitzung](private-sitzung-aendern.md) auf `False` gesetzt sein. Die beiden Einstellungen schließen sich gegenseitig aus.
::::
