# So setzt du das Admin Passwort auf deinem Renown Server

Mit dem Admin Passwort kannst du dich im Spiel als Admin anmelden und Admin-Befehle ausführen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Admin Passwort setzen</b><br>
   Trage das gewünschte Passwort im Feld **Admin Passwort** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: warning Achtung
Halte das Admin Passwort geheim und teile es ausschließlich mit Personen, denen du vollen Zugriff auf den Server geben möchtest.
::::
