# So trittst du deinem Renown Server bei

1. <b>Renown starten</b><br>
   Starte Renown.

2. <b>Spielen auswählen</b><br>
   Klicke im Hauptmenü auf **Spielen**.

3. <b>Region auswählen</b><br>
   Wähle deine Region aus.

4. <b>Alle anzeigen</b><br>
   Klicke auf **Alle**, um alle verfügbaren Server anzuzeigen.

5. <b>Server suchen</b><br>
   Suche nach dem Namen deines Servers im Suchfeld.

6. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Beitreten**. Falls dein Server passwortgeschützt ist, gib das Passwort ein und bestätige.

:::: tip Tipp
Damit dein Server in der öffentlichen Serverliste angezeigt wird, muss [Öffentlicher Server](oeffentlicher-server-aendern.md) auf `True` und [Private Sitzung](private-sitzung-aendern.md) auf `False` gesetzt sein.
::::
