# So aktivierst du die Private Sitzung auf deinem Renown Server

Mit dieser Einstellung legst du fest, ob dein Server als private Sitzung läuft.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Private Sitzung anpassen</b><br>
   Setze das Feld **Private Sitzung** auf den gewünschten Wert:
   - `True` — Private Sitzung ist aktiviert
   - `False` — Private Sitzung ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: warning Wichtig
Wenn **Private Sitzung** auf `False` gesetzt ist, muss [Öffentlicher Server](oeffentlicher-server-aendern.md) auf `True` gesetzt sein. Die beiden Einstellungen schließen sich gegenseitig aus.
::::
