# So aktivierst du First Person Only auf deinem Renown Server

Wenn aktiviert, ist auf deinem Server ausschließlich die First-Person-Ansicht verfügbar. Spieler können nicht in die Third-Person-Ansicht wechseln.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>First Person Only anpassen</b><br>
   Setze das Feld **First Person Only** auf den gewünschten Wert:
   - `1` — First Person Only ist aktiviert
   - `0` — First Person Only ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
