# So passt du die Map Rotation auf deinem Renown Server an

Die Map Rotation legt fest, welche Maps während des Spiels in welcher Reihenfolge rotiert werden.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Map Rotation anpassen</b><br>
   Trage die gewünschten Maps im Feld **Map Rotation** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: tip Tipp
Damit der Server beim Start die erste Map aus der Rotation lädt, kannst du zusätzlich die Option [Use First Map](use-first-map-aktivieren.md) aktivieren.
::::
