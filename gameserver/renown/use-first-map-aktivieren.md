# So aktivierst du Use First Map auf deinem Renown Server

Wenn aktiviert, lädt dein Server beim Start immer die erste Map aus der [Map Rotation](map-rotation-aendern.md), anstatt sie zufällig oder anhand des letzten Spielzustands zu wählen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Use First Map anpassen</b><br>
   Setze das Feld **Use First Map** auf den gewünschten Wert:
   - `True` — Use First Map ist aktiviert
   - `False` — Use First Map ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
