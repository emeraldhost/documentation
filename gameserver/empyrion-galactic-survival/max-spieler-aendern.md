# So änderst du die maximale Spielerzahl deines Empyrion Servers

Du kannst festlegen, wie viele Spieler gleichzeitig auf deinem Server spielen können.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Maximale Spieler ändern</b><br>
   Trage den gewünschten Wert im Feld **Maximale Spieler** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
