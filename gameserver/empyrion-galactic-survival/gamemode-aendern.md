# So änderst du den Gamemode deines Empyrion Servers

Du kannst festlegen, in welchem Spielmodus dein Server läuft.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Gamemode ändern</b><br>
   Trage den gewünschten Wert im Feld **Gamemode** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Info
Typische Werte sind `Survival` oder `Creative`.
::::
