# So setzt du ein Server Passwort auf deinem Empyrion Server

Du kannst deinen Server mit einem Passwort schützen, sodass nur Spieler mit dem Passwort beitreten können.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Passwort setzen</b><br>
   Trage das gewünschte Passwort im Feld **Server Passwort** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: tip Tipp
Um das Passwort zu entfernen und den Server wieder öffentlich zugänglich zu machen, lasse das Feld leer und speichere die Einstellung.
::::
