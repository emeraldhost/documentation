# So änderst du den Savegame-Namen deines Empyrion Servers

Du kannst festlegen, welcher Spielstand von deinem Server geladen wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Savegame-Namen ändern</b><br>
   Trage den gewünschten Wert im Feld **Save Game** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Info
Dieser Wert bestimmt, welcher Savegame-Ordner geladen wird. Durch Ändern wird ein anderer Spielstand geladen oder ein neuer erstellt.
::::
