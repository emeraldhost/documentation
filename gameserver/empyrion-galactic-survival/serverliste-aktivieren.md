# So aktivierst du die Serverliste auf deinem Empyrion Server

Du kannst festlegen, ob dein Server öffentlich in der Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Serverliste aktivieren</b><br>
   Aktiviere den Schalter bei **Serverliste** und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Info
Wenn aktiviert, erscheint dein Server im Steam-Serverbrowser.
::::
