# So trittst du deinem Empyrion: Galactic Survival Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct IP and Port

1. <b>Empyrion starten</b><br>
   Starte Empyrion: Galactic Survival über Steam.

2. <b>Join a Game öffnen</b><br>
   Klicke im Hauptmenü auf **Join a Game**.

3. <b>Direct IP and port eingeben</b><br>
   Trage im Feld **Direct IP and port** die IP-Adresse und den Port deines Servers ein:

   :::: tip Beispiel
   ```
   123.45.67.89:30000
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Connect**, um dem Server beizutreten.

## Über den Steam Server-Browser

Alternativ kannst du deinen Server über den Steam Server-Browser hinzufügen.

1. <b>Steam Server-Browser öffnen</b><br>
   Öffne Steam und gehe zu **Anzeigen → Server**.

2. <b>Favoriten öffnen</b><br>
   Wechsle zum Tab **Favoriten** und klicke unten auf **Server hinzufügen**.

3. <b>Server-Adresse eingeben</b><br>
   Trage die IP-Adresse und den Query-Port deines Servers ein und bestätige.

4. <b>Beitreten</b><br>
   Wähle deinen Server aus den Favoriten und klicke auf **Verbinden**.
