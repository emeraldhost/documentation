# So änderst du die Server Beschreibung deines Empyrion Servers

Du kannst die Beschreibung deines Servers anpassen, die Spielern in der Serverliste angezeigt wird.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Beschreibung ändern</b><br>
   Trage den gewünschten Text im Feld **Server Beschreibung** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
