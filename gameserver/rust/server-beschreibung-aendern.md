# So änderst du die Server Beschreibung deines Rust Servers

Die Server Beschreibung wird angezeigt, wenn Spieler deinen Server im Server-Browser anklicken.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Beschreibung anpassen</b><br>
   Trage den gewünschten Wert im Feld **Server Beschreibung** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
