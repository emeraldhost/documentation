# So trittst du deinem Rust Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über die F1-Konsole

1. <b>Rust starten</b><br>
   Starte Rust über Steam.

2. <b>Konsole öffnen</b><br>
   Drücke im Hauptmenü die Taste `F1`, um die Konsole zu öffnen.

3. <b>Connect-Befehl ausführen</b><br>
   Gib folgenden Befehl ein und drücke Enter:

   ```
   client.connect 123.45.67.89:28015
   ```

   Ersetze die IP-Adresse und den Port durch die deines Servers.

## Über die Steam-Favoriten

Alternativ kannst du deinen Server als Steam-Favorit hinzufügen, damit er dauerhaft in deiner Liste erscheint.

1. <b>Steam Server-Browser öffnen</b><br>
   Öffne Steam und gehe zu **Anzeigen → Server**.

2. <b>Favoriten öffnen</b><br>
   Wechsle zum Tab **Favoriten** und klicke unten auf **Server hinzufügen**.

3. <b>Server-Adresse eingeben</b><br>
   Trage die IP-Adresse und den Port deines Servers ein und klicke auf **Hinzufügen zu Favoriten**.

4. <b>In Rust beitreten</b><br>
   Starte Rust, klicke auf **Play Game** und wechsle zum Tab **Favoriten**. Wähle deinen Server aus und klicke auf **Beitreten**.
