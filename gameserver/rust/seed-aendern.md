# So änderst du den Seed deines Rust Servers

Mit dem Seed bestimmst du, welche Welt auf deinem Server generiert wird.

:::: warning Achtung
Das Ändern des Seeds auf einem bestehenden Savegame hat keinen Effekt — die Änderung greift erst, wenn eine neue Map generiert wird.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Seed anpassen</b><br>
   Trage den gewünschten Wert im Feld **Welt Seed** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
