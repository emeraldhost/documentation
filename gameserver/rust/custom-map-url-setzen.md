# So setzt du eine Custom Map URL auf deinem Rust Server

Mit einer Custom Map URL kannst du eine eigene Map auf deinem Server verwenden.

:::: info Hinweis
Ermöglicht die Nutzung einer eigenen Map (.map-Datei), die online gehostet wird. Lasse das Feld leer, um eine prozedurale Map zu verwenden.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Custom Map URL anpassen</b><br>
   Trage den gewünschten Wert im Feld **Custom Map URL** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
