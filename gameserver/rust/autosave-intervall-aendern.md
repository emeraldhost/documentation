# So änderst du das Autosave-Intervall deines Rust Servers

Mit dem Autosave-Intervall legst du fest, wie oft dein Server automatisch gespeichert wird.

:::: info Hinweis
Legt das Intervall für das automatische Speichern des Servers in Sekunden fest. Standard: 60.
::::

:::: tip Tipp
Niedrigere Werte speichern häufiger, können aber kurze Lag-Spikes verursachen.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Autosave-Intervall anpassen</b><br>
   Trage den gewünschten Wert in Sekunden im Feld **Automatisch Speichern** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
