# So setzt du die Server URL deines Rust Servers

Mit der Server URL hinterlegst du einen Link zu deiner Website, der Spielern im Spiel angezeigt wird.

:::: info Hinweis
Die URL für Deinen Server. Diese wird angezeigt, wenn Spieler auf die Schaltfläche 'Website besuchen' klicken.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server URL setzen</b><br>
   Trage den gewünschten Wert im Feld **Server URL** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
