# So setzt du das Server Image deines Rust Servers

Mit dem Server Image hinterlegst du ein Bild, das oberhalb deiner Serverliste angezeigt wird.

:::: info Hinweis
Das Bild für den Anfang Deiner Serverliste. Erwartet wird eine URL zu einem Bild.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Image setzen</b><br>
   Trage die gewünschte Bild-URL im Feld **Server Image** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
