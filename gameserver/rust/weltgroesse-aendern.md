# So änderst du die Weltgröße deines Rust Servers

Die Weltgröße bestimmt, wie groß die generierte Welt auf deinem Server ist.

:::: info Hinweis
Übliche Werte liegen zwischen 1000 (klein) und 6000 (groß). Größere Maps benötigen mehr RAM und längere Ladezeiten.
::::

:::: warning Achtung
Die Änderung greift erst, wenn eine neue Map generiert wird.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Weltgröße anpassen</b><br>
   Trage den gewünschten Wert im Feld **Weltgröße** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
