# So änderst du die Map deines Rust Servers

Mit der Map legst du fest, welche Welt Rust auf deinem Server verwenden soll.

:::: info Hinweis
Die Welt, die Rust verwenden soll. Übliche Werte sind `Procedural Map`, `Barren` oder ein konkreter Map-Seed bzw. Name. Standard: `Procedural Map`.
::::

:::: warning Achtung
Die Änderung greift erst, wenn eine neue Map generiert wird.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Map anpassen</b><br>
   Trage den gewünschten Wert im Feld **Map** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
