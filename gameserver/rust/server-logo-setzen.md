# So setzt du das Server Logo deines Rust Servers

Mit dem Server Logo hinterlegst du ein kreisförmiges Bild für die Rust+-App.

:::: info Hinweis
Das kreisförmige Server Logo für die Rust+-App. Erwartet wird eine URL zu einem Bild.
::::

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Logo setzen</b><br>
   Trage die gewünschte Bild-URL im Feld **Server Logo** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
