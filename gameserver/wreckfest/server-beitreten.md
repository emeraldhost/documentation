# So trittst du deinem Wreckfest Server bei

## Verbindungsdaten finden

:::: warning Wichtig
Für die Verbindung benötigst du die **IP-Adresse** und den **Query Port** deines Servers. Den Query Port findest du in den **Einstellungen** deines Servers in der Verwaltung. Verwende **nicht** den Gameport, sondern den Query Port!
::::

## Über den Server-Browser

1. <b>Wreckfest starten</b><br>
   Starte Wreckfest.

2. <b>Multiplayer öffnen</b><br>
   Wähle im Hauptmenü **Multiplayer**.

3. <b>Server suchen</b><br>
   Suche nach dem Namen deines Servers im Suchfeld oder filtere die Liste.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Beitreten**. Falls dein Server passwortgeschützt ist, gib das Passwort ein und bestätige.

## Über die Steam-Favoriten

Wreckfest greift auf die Steam-Server-Favoriten zurück. Du kannst deinen Server dort hinzufügen, damit du ihn schneller findest.

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann unten rechts auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den **Query Port** deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>Wreckfest starten</b><br>
   Starte Wreckfest und öffne den **Multiplayer**-Bereich. Dein Server wird nun bevorzugt angezeigt.
