# So änderst du die Willkommensnachricht deines Wreckfest Servers

Die Willkommensnachricht wird Spielern beim Beitritt zum Server angezeigt. Sie eignet sich für Regeln, Hinweise oder einen Gruß.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Willkommensnachricht ändern</b><br>
   Trage den gewünschten Text im Feld **Willkommensnachricht** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.
