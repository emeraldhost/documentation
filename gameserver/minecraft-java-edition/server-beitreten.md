# So trittst du deinem Minecraft Java Edition Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Minecraft starten</b><br>
   Starte Minecraft Java Edition.

2. <b>Multiplayer öffnen</b><br>
   Klicke im Hauptmenü auf **Mehrspieler**.

3. <b>Server hinzufügen</b><br>
   Klicke auf **Server hinzufügen**.

4. <b>Server-Daten eingeben</b><br>
   Gib einen Namen für deinen Server und die Server-Adresse im Format `IP:Port` ein.

   :::: tip Beispiel
   ```
   123.45.67.89:25565
   ```
   ::::

5. <b>Fertig</b><br>
   Klicke auf **Fertig**. Dein Server erscheint nun in der Liste und du kannst ihm beitreten.

:::: tip Tipp
Wenn du eine Domain mit deinem Server verknüpft hast, kannst du diese statt der IP-Adresse verwenden. Siehe [Domain mit Minecraft Server verknüpfen](../../domains/dns-eintraege/minecraft-server.md).
::::
