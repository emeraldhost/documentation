# So erhöhst du die Slots auf einem Minecraft Java Server

1. <b>Datei öffnen</b><br>
   Verbinde dich per [SFTP](../sftp-verbindung-herstellen.md) mit deinem Server und öffne die Datei `server.properties`.

2. <b>Eintrag suchen</b><br>
   Suche den folgenden Eintrag:

   ```
   max-players=20
   ```

3. <b>Slots ändern</b><br>
   Ändere den Wert auf die gewünschte Anzahl an Spielern.

4. <b>Server neu starten</b><br>
   Speichere die Änderungen und starte den Server neu.

:::: warning Warnung
Stelle sicher, dass dein Server genügend Ressourcen (wie RAM und CPU) zur Verfügung hat, um die erhöhte Spieleranzahl zu unterstützen.
::::
