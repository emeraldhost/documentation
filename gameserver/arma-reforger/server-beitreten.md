# So trittst du deinem Arma Reforger Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über den Servernamen suchen

1. <b>Arma Reforger starten</b><br>
   Starte Arma Reforger.

2. <b>Multiplayer öffnen</b><br>
   Klicke im Hauptmenü auf **Multiplayer**.

3. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers in der Server-Liste.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Join**.

## Über Direct Connect

1. <b>Arma Reforger starten</b><br>
   Starte Arma Reforger.

2. <b>Multiplayer öffnen</b><br>
   Klicke im Hauptmenü auf **Multiplayer**.

3. <b>Direct Join</b><br>
   Klicke auf **Direct Join** und gib die IP-Adresse sowie den Port deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:2001
   ```
   ::::

4. <b>Verbinden</b><br>
   Klicke auf **Join**, um dem Server beizutreten.
