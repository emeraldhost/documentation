# So trittst du deinem Euro Truck Simulator 2 Server bei

## Verbindungsdaten finden

:::: info Hinweis
Du benötigst die **Search ID** und das **Server Passwort** deines Servers. Die **Search ID** wird in der **Server-Konsole** angezeigt, sobald der Server gestartet ist. Das **Server Passwort** findest du in der **Verwaltung** deines Servers.
::::

## Über die Convoy-Suche

1. <b>Euro Truck Simulator 2 starten</b><br>
   Starte Euro Truck Simulator 2.

2. <b>Convoy öffnen</b><br>
   Klicke im Hauptmenü auf **Convoy**.

3. <b>Server suchen</b><br>
   Wähle **Search Convoy** und gib die **Search ID** deines Servers ein.

4. <b>Passwort eingeben</b><br>
   Gib das **Server Passwort** ein.

5. <b>Verbinden</b><br>
   Klicke auf **Join**, um dem Server beizutreten.
