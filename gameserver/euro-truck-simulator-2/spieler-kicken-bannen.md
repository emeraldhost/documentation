# So kickst und bannst du Spieler auf deinem Euro Truck Simulator 2 Server

Du kannst Spieler über das Ingame-Convoy-Menü aus deiner Session kicken.

## Spieler kicken

1. <b>Convoy-Menü öffnen</b><br>
   Öffne das Convoy-Menü während du auf dem Server spielst.

2. <b>Spieler auswählen</b><br>
   Wähle den Spieler aus der Spielerliste aus, den du kicken möchtest.

3. <b>Spieler kicken</b><br>
   Klicke auf die Option zum Kicken des Spielers. Der Spieler wird aus der Session entfernt.

:::: info Hinweis
Gekickte Spieler können der Session erneut beitreten. Eine dauerhafte Bann-Funktion ist in Euro Truck Simulator 2 nicht verfügbar.
::::
