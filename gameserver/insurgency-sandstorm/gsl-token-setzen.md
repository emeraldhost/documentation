# So setzt du den GSL Token auf deinem Insurgency: Sandstorm Server

Der GSL Token (Game Server Login Token) wird benötigt, damit dein Server in der Serverliste angezeigt wird.

## Token generieren

1. <b>Steam Game Server Seite öffnen</b><br>
   Öffne [https://steamcommunity.com/dev/managegameservers](https://steamcommunity.com/dev/managegameservers) und melde dich mit deinem Steam-Konto an.

2. <b>Token erstellen</b><br>
   Erstelle einen neuen Token mit der App-ID **581320** (Insurgency: Sandstorm) und kopiere ihn.

## Token hinterlegen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Token eintragen</b><br>
   Füge den Token im Feld **GSL Token** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Zusätzlich wird ein [Game Stats Token](game-stats-token-setzen.md) benötigt, damit dein Server in der Serverliste erscheint.
::::
