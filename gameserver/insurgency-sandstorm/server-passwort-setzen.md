# So setzt du ein Passwort auf deinem Insurgency: Sandstorm Server

Du kannst deinen Server mit einem Passwort schützen, sodass nur Spieler mit dem Passwort beitreten können. Das ist besonders sinnvoll, wenn dein Server über die [Server Tokens](game-stats-token-setzen.md) in der Serverliste auftaucht und du trotzdem nicht möchtest, dass beliebige Spieler beitreten.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Passwort setzen</b><br>
   Trage im Feld **Server Passwort** das gewünschte Passwort ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: tip Tipp
Um das Passwort zu entfernen und den Server wieder öffentlich zu machen, lösche den Eintrag im Passwort-Feld und speichere.
::::
