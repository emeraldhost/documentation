# So trittst du deinem Insurgency: Sandstorm Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über den Community-Server-Browser

1. <b>Insurgency: Sandstorm starten</b><br>
   Starte Insurgency: Sandstorm.

2. <b>Server-Browser öffnen</b><br>
   Wähle im Hauptmenü **Play** und anschließend **Community Servers**.

3. <b>Server suchen</b><br>
   Suche nach dem Namen deines Servers oder filtere die Liste, z.B. nach Map oder Spielmodus.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und tritt ihm bei. Falls dein Server passwortgeschützt ist, gib das Passwort ein.

## Über die Konsole (Direct Connect)

1. <b>Konsole öffnen</b><br>
   Drücke im Hauptmenü die Taste `~` (Tilde), um die Konsole zu öffnen.

2. <b>Verbindungsbefehl eingeben</b><br>
   Gib folgenden Befehl ein und ersetze IP und Port durch deine Verbindungsdaten:

   ```
   open IP-Adresse:Port
   ```

   :::: tip Beispiel
   ```
   open 123.45.67.89:27102
   ```
   ::::

3. <b>Verbinden</b><br>
   Bestätige mit `Enter`, um dem Server beizutreten.

:::: info Hinweis
Damit dein Server im Community-Server-Browser angezeigt wird, müssen [Game Stats Token](game-stats-token-setzen.md) und [GSL Token](gsl-token-setzen.md) hinterlegt sein.
::::
