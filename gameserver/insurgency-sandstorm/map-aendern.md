# So änderst du die Map auf deinem Insurgency: Sandstorm Server

Die Map legt fest, welche Karte beim Start des Servers geladen wird (z.B. `Ministry`, `Refinery`, `Hideout`).

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Map ändern</b><br>
   Trage im Feld **Map** den Namen der gewünschten Karte ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Die Map muss zum gewählten [Szenario](szenario-aendern.md) passen. Eine vollständige Liste der Karten und Szenarien findest du im offiziellen [Server Admin Guide](https://mod.io/g/insurgencysandstorm/r/server-admin-guide).
::::
