# So setzt du den Game Stats Token auf deinem Insurgency: Sandstorm Server

Der Game Stats Token wird benötigt, damit dein Server in der Serverliste angezeigt wird und Statistiken erfasst werden.

## Token generieren

1. <b>Game Stats Seite öffnen</b><br>
   Öffne [https://gamestats.sandstorm.game/](https://gamestats.sandstorm.game/) und melde dich an.

2. <b>Token erstellen</b><br>
   Erstelle einen neuen Token für deinen Server und kopiere ihn.

## Token hinterlegen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Token eintragen</b><br>
   Füge den Token im Feld **Game Stats Token** ein und speichere die Einstellung.

4. <b>Server neu starten</b><br>
   Starte deinen Server neu, damit die Änderung übernommen wird.

:::: info Hinweis
Zusätzlich wird ein [GSL Token](gsl-token-setzen.md) benötigt, damit dein Server in der Serverliste erscheint.
::::
