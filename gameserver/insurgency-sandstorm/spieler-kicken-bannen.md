# So kickst und bannst du Spieler auf deinem Insurgency: Sandstorm Server

Stör- oder Regelbrecher kannst du als Admin direkt im Spiel über das Admin-Menü kicken oder dauerhaft bannen.

## Voraussetzung

Du benötigst Admin-Rechte auf dem Server. Wie du dich als Admin einträgst, erfährst du unter [Admins hinzufügen](admins-hinzufuegen.md).

## Spieler kicken oder bannen

1. <b>Server beitreten</b><br>
   Tritt deinem Server bei.

2. <b>Admin-Menü öffnen</b><br>
   Öffne das Admin-Menü mit der zugewiesenen Taste — standardmäßig die **Minus-Taste** (`-`) des Nummernblocks.

3. <b>Spielerliste öffnen</b><br>
   Wechsle zum Tab **Players**.

4. <b>Spieler auswählen</b><br>
   Wähle den gewünschten Spieler aus und klicke auf **Kick** oder **Ban**.

:::: tip Tipp
Für eine Selbstmoderation ohne feste Admins kannst du zusätzlich das [Spieler-Voting](spieler-voting-konfigurieren.md) aktivieren, mit dem Spieler problematische Mitspieler per Abstimmung entfernen können.
::::
