# So trittst du deinem VEIN Server bei

## Verbindungsdaten finden

:::: warning Wichtig
Für die Verbindung benötigst du die **IP-Adresse** und den **Query Port** deines Servers. Den Query Port findest du in den **Einstellungen** deines Servers in der Verwaltung. Verwende **nicht** den Gameport, sondern den Query Port!
::::

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann unten rechts auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den **Query Port** deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>VEIN starten</b><br>
   Starte VEIN.

6. <b>Server suchen</b><br>
   Wähle im Hauptmenü **Multiplayer** und filtere nach **Favoriten** — dein Server sollte nun in der Liste angezeigt werden.

7. <b>Server beitreten</b><br>
   Wähle deinen Server aus und klicke auf **Join**.

## Über den Servernamen suchen

1. <b>VEIN starten</b><br>
   Starte VEIN und wähle im Hauptmenü **Multiplayer**.

2. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers.

3. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Join**.
