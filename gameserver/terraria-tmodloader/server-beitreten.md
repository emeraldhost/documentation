# So trittst du deinem Terraria: tModLoader Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>tModLoader starten</b><br>
   Starte Terraria: tModLoader.

2. <b>Charakter auswählen</b><br>
   Klicke auf **Multiplayer** und wähle deinen Charakter aus.

3. <b>Mit IP verbinden</b><br>
   Wähle **Join via IP**.

4. <b>Server-Daten eingeben</b><br>
   Gib die IP-Adresse und den Port deines Servers ein.

   :::: tip Beispiel
   ```
   IP: 123.45.67.89
   Port: 7777
   ```
   ::::

5. <b>Verbinden</b><br>
   Klicke auf **Accept**, um dem Server beizutreten.

:::: warning Achtung
Stelle sicher, dass du **dieselben Mods** wie der Server installiert hast, sonst kannst du nicht beitreten.
::::
