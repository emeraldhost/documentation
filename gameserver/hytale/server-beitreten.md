# So trittst du deinem Hytale Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Hytale starten</b><br>
   Öffne den Hytale Launcher und starte das Spiel.

2. <b>Server-Menü öffnen</b><br>
   Klicke im Hauptmenü auf **Servers**.

3. <b>Direct Connect wählen</b><br>
   Klicke unten rechts auf **Direct Connect**.

4. <b>IP-Adresse eingeben</b><br>
   Gib die IP-Adresse deines Servers ein und klicke auf **Connect**.

## Server speichern

Um den Server für zukünftige Verbindungen zu speichern:

1. <b>Server hinzufügen</b><br>
   Klicke im Server-Menü unten rechts auf **Add Server**.

2. <b>Server-Details eingeben</b><br>
   Gib die IP-Adresse in das Feld **Connection Address** ein und vergib einen Namen.

3. <b>Speichern</b><br>
   Klicke auf **Add Server**. Der Server erscheint nun in deiner Server-Liste.

:::: tip Tipp
Falls der Server nicht sofort erreichbar ist, warte einen Moment und versuche es erneut. Server benötigen manchmal einige Sekunden zum Aufwachen.
::::
