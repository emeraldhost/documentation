# So trittst du deinem Minecraft Bedrock Edition Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über Direct Connect

1. <b>Minecraft starten</b><br>
   Starte Minecraft Bedrock Edition.

2. <b>Spielen</b><br>
   Klicke auf **Spielen**.

3. <b>Server-Tab öffnen</b><br>
   Wähle den Tab **Server**.

4. <b>Server hinzufügen</b><br>
   Scrolle nach unten und klicke auf **Server hinzufügen**.

5. <b>Server-Daten eingeben</b><br>
   Gib einen Namen für deinen Server, die IP-Adresse und den Port ein.

   :::: tip Beispiel
   ```
   IP: 123.45.67.89
   Port: 19132
   ```
   ::::

6. <b>Speichern</b><br>
   Klicke auf **Speichern**. Dein Server erscheint nun in der Liste und du kannst ihm beitreten.

## Auf Konsole (Xbox, PlayStation, Switch)

Konsolen erlauben es nicht, eigene Server-IPs direkt hinzuzufügen. Über einen DNS-Trick (BedrockConnect) kannst du die Liste der "Featured Server" so umleiten, dass du deine eigene IP eingeben kannst.

:::: warning Achtung
Du änderst hierbei deine DNS-Einstellungen. Notiere dir vorher die aktuellen Werte, falls du sie später wiederherstellen möchtest.
::::

### DNS-Adressen

| Konsole | Primär | Sekundär |
|---------|--------|----------|
| Xbox One / Series X\|S | `104.238.130.180` | `8.8.8.8` |
| PlayStation 4 / 5 | `45.55.68.52` | `8.8.8.8` |
| Nintendo Switch | `104.238.130.180` | `8.8.8.8` |

### Xbox

1. <b>Netzwerk-Einstellungen öffnen</b><br>
   Gehe zu **Einstellungen → Allgemein → Netzwerkeinstellungen → Erweiterte Einstellungen → DNS-Einstellungen**.

2. <b>Manuell wählen</b><br>
   Wähle **Manuell** und trage die DNS-Adressen aus der Tabelle ein.

3. <b>Speichern und neustarten</b><br>
   Speichere die Einstellung und starte die Xbox neu.

### PlayStation

1. <b>Netzwerk-Einstellungen öffnen</b><br>
   Gehe zu **Einstellungen → Netzwerk → Einstellungen → Internet einrichten**.

2. <b>Verbindung anpassen</b><br>
   Wähle dein Netzwerk aus, öffne die **Erweiterten Einstellungen** und setze die **DNS-Einstellungen** auf **Manuell**.

3. <b>DNS eintragen</b><br>
   Trage die DNS-Adressen aus der Tabelle ein und bestätige.

4. <b>PlayStation neustarten</b><br>
   Starte deine Konsole neu.

### Server verbinden

1. <b>Minecraft starten</b><br>
   Öffne Minecraft Bedrock Edition und wechsle in den Tab **Server**.

2. <b>Featured Server beitreten</b><br>
   Klicke auf einen der angezeigten Featured Server (z.B. Mineville, Lifeboat). Du wirst stattdessen zu BedrockConnect weitergeleitet.

3. <b>Eigene IP eingeben</b><br>
   Wähle **Connect to a Server** und gib die IP-Adresse sowie den Port deines Servers ein.
