# So legst du den Server Owner auf deinem Astroneer Server fest

Über den Server Owner legst du fest, welcher Steam-Account Administrator-Rechte auf dem Server erhält. Du musst dafür sowohl den Steam-Anzeigenamen als auch die Steam64 ID eintragen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Server Owner Name eintragen</b><br>
   Trage im Feld **Server Owner Name** den Steam-Anzeigenamen des Besitzers ein.

4. <b>Server Owner GUID eintragen</b><br>
   Trage im Feld **Server Owner GUID** die [SteamID64](../steamid64-herausfinden.md) des Besitzers ein.

5. <b>Server neu starten</b><br>
   Speichere die Einstellungen und starte deinen Server neu.

:::: info Hinweis
Die Admin-Rechte werden der ersten Person zugewiesen, die sich mit dem angegebenen Namen verbindet.
::::
