# So hinterlegst du den Lizenz Key auf deinem FiveM Server

Um deinen FiveM Server korrekt starten zu können, musst du einen **Lizenz Key** in den Einstellungen hinterlegen. Ohne diesen Key wird dein Server nicht starten.

## Lizenz Key erstellen

1. <b>Cfx.re Portal öffnen</b><br>
   Öffne das offizielle [Cfx.re Portal](https://portal.cfx.re/).

2. <b>Anmelden</b><br>
   Logge dich mit deinem Cfx.re Account ein oder erstelle einen neuen.

3. <b>Key generieren</b><br>
   Erstelle einen neuen Lizenz Key für deinen Server und kopiere ihn.

   :::: info Hinweis
   Beim Erstellen des Keys wirst du nach der **IP-Adresse** deines Servers gefragt. Diese findest du in der Verwaltung deines Servers.
   ::::

## Lizenz Key hinterlegen

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Lizenz Key eintragen</b><br>
   Trage den generierten Key im Feld **Lizenz Key** ein.

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: warning Achtung
Teile deinen Lizenz Key niemals mit anderen Personen. Jeder Server benötigt einen eigenen, einzigartigen Key.
::::
