# So trittst du deinem FiveM Server bei

## Verbindungsdaten finden

:::: info Hinweis
Die IP-Adresse und den Port deines Servers findest du in der **Verwaltung** deines Servers unter der Übersicht.
::::

## Über den Servernamen suchen

1. <b>FiveM starten</b><br>
   Starte FiveM.

2. <b>Server-Browser öffnen</b><br>
   Klicke im Hauptmenü auf **Play**.

3. <b>Server suchen</b><br>
   Nutze den Filter und suche nach dem Namen deines Servers in der Server-Liste.

4. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Connect**.

## Über Direct Connect

1. <b>FiveM starten</b><br>
   Starte FiveM.

2. <b>Konsole öffnen</b><br>
   Drücke `F8`, um die Konsole zu öffnen.

3. <b>Verbinden</b><br>
   Gib folgenden Befehl ein:

   ```
   connect IP:Port
   ```

   :::: tip Beispiel
   ```
   connect 123.45.67.89:30120
   ```
   ::::
