# So aktivierst oder deaktivierst du VoN auf deinem DayZ Server

Mit **Voice over Network (VoN)** können Spieler im Spiel über ihr Mikrofon mit anderen Spielern kommunizieren. Du kannst die Funktion über die Verwaltung aktivieren oder deaktivieren.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>VoN umschalten</b><br>
   Aktiviere oder deaktiviere die Option **VoN deaktivieren**:

   - **Aktiviert**: Spieler können nicht über das Spiel sprechen
   - **Deaktiviert**: Spieler können über VoN miteinander kommunizieren

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
VoN funktioniert nur in einer bestimmten Reichweite — nahe Spieler hören sich, weit entfernte nicht. So entsteht ein realistisches Kommunikationserlebnis.
::::
