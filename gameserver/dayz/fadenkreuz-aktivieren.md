# So aktivierst oder deaktivierst du das Fadenkreuz auf deinem DayZ Server

Du kannst das Fadenkreuz über die Verwaltung aktivieren oder deaktivieren.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Fadenkreuz umschalten</b><br>
   Aktiviere oder deaktiviere die Option **Fadenkreuz deaktivieren**:

   - **Aktiviert**: Spieler haben kein Fadenkreuz und müssen die Iron-Sights oder Zielfernrohre nutzen
   - **Deaktiviert**: Spieler haben ein Fadenkreuz auf dem Bildschirm

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: tip Tipp
Server ohne Fadenkreuz gelten als realistischer und werden oft mit reinen First-Person-Servern kombiniert.
::::
