# So änderst du die Tageszeit auf deinem DayZ Server

Du kannst die Geschwindigkeit der Ingame-Zeit und die Helligkeit der Nächte über die Verwaltung anpassen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Zeitmultiplikator anpassen</b><br>
   Trage im Feld **Zeitmultiplikator** den gewünschten Wert ein (Bereich: `0` - `24`).

   :::: tip Beispiel
   Ein Wert von `12` bedeutet, dass ein vollständiger 24-Stunden-Tag im Spiel **12-mal schneller** abläuft, also nur 2 Stunden in Echtzeit dauert.
   ::::

4. <b>Dunklere Nächte einstellen (optional)</b><br>
   Aktiviere die Option **Dunklere Nächte**, um realistischere und dunklere Nächte zu erhalten. Spieler benötigen dann eine Lichtquelle, um sich orientieren zu können.

5. <b>Server neu starten</b><br>
   Speichere die Einstellungen und starte deinen Server neu.
