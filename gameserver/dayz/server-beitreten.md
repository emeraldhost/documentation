# So trittst du deinem DayZ Server bei

Du kannst deinem Server über den Steam-Server-Browser oder über DayZ direkt beitreten.

## Über den Steam-Server-Browser

1. <b>Steam öffnen</b><br>
   Öffne den Steam-Client.

2. <b>Server-Browser öffnen</b><br>
   Klicke oben links auf **Anzeige** und wähle **Spielserver**.

3. <b>Server hinzufügen</b><br>
   Klicke auf den Tab **Favoriten** und dann auf **+**.

4. <b>Serveradresse eintragen</b><br>
   Gib die IP-Adresse und den **Query Port** deines Servers ein.

   :::: tip Beispiel
   ```
   123.45.67.89:27016
   ```
   ::::

5. <b>Verbinden</b><br>
   Wähle den Server aus und klicke auf **Verbinden**.

## Über DayZ ingame

1. <b>DayZ starten</b><br>
   Starte DayZ und wähle im Hauptmenü **Spielen**.

2. <b>Community-Server öffnen</b><br>
   Wähle den Tab **Community-Server**.

3. <b>Server suchen</b><br>
   Suche deinen Server über den Servernamen oder filtere nach IP-Adresse.

4. <b>Server beitreten</b><br>
   Wähle deinen Server und klicke auf **Spielen**.

:::: warning Wichtig
Für die Verbindung benötigst du den **Query Port**, **nicht** den Gameport. Den Query Port findest du in den **Einstellungen** deines Servers in der Verwaltung.
::::
