# So aktivierst du Demo-Aufzeichnungen auf deinem Squad 44 Server

Demos sind Aufnahmen von Matches, die du nachträglich anschauen oder zur Überprüfung von Vorfällen verwenden kannst.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Demos aufzeichnen anpassen</b><br>
   Setze das Feld **Demos aufzeichnen** auf den gewünschten Wert:
   - `true` — Demo-Aufzeichnung ist aktiviert
   - `false` — Demo-Aufzeichnung ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Demo-Dateien können mit der Zeit viel Speicherplatz belegen. Lösche oder lade alte Demos regelmäßig herunter, falls dein Speicherplatz knapp wird.
::::
