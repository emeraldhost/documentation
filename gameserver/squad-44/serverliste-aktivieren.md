# So aktivierst du die Serverliste auf deinem Squad 44 Server

Diese Einstellung legt fest, ob dein Server im öffentlichen Custom Browser angezeigt wird oder nicht.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Serverliste anpassen</b><br>
   Setze das Feld **Serverliste** auf den gewünschten Wert:
   - `true` — Serverliste ist aktiviert (dein Server wird im Custom Browser angezeigt)
   - `false` — Serverliste ist deaktiviert (dein Server wird nicht angezeigt)

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.

:::: info Hinweis
Wenn die Serverliste deaktiviert ist, können Spieler deinen Server im Custom Browser nicht finden. Sie müssen den Server stattdessen über andere Methoden kontaktieren.
::::
