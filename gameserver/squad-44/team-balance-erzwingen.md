# So erzwingst du Team Balance auf deinem Squad 44 Server

Mit dieser Einstellung sorgt der Server automatisch dafür, dass die Teams gleichmäßig stark besetzt sind.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Team Balance erzwingen anpassen</b><br>
   Setze das Feld **Team Balance erzwingen** auf den gewünschten Wert:
   - `true` — Team Balance erzwingen ist aktiviert
   - `false` — Team Balance erzwingen ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
