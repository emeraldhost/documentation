# So erlaubst du den Teamwechsel auf deinem Squad 44 Server

Du kannst festlegen, ob Spieler während eines Spiels das Team wechseln dürfen.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Teamwechsel erlauben anpassen</b><br>
   Setze das Feld **Teamwechsel erlauben** auf den gewünschten Wert:
   - `true` — Teamwechsel ist aktiviert
   - `false` — Teamwechsel ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
