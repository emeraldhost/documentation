# So trittst du deinem Squad 44 Server bei

Squad 44 bietet keinen direkten Verbindungsaufbau über die IP-Adresse — du trittst deinem Server immer über den **Custom Browser** im Spiel bei.

## Voraussetzung

Notiere dir den **Servernamen** deines Servers. Diesen kannst du in der Verwaltung unter **Einstellungen** anpassen — siehe [Server Name ändern](server-name-aendern.md).

## Server beitreten

1. <b>Squad 44 starten</b><br>
   Starte Squad 44.

2. <b>Server-Browser öffnen</b><br>
   Wähle im Hauptmenü **Server Browser**.

3. <b>Custom Browser öffnen</b><br>
   Wechsle in den Tab **Custom Browser**.

4. <b>Server suchen</b><br>
   Gib den Namen deines Servers in das Suchfeld ein.

5. <b>Server beitreten</b><br>
   Wähle deinen Server aus der Liste und klicke auf **Join Server**. Falls dein Server passwortgeschützt ist, gibst du das Passwort ein und bestätigst.

:::: tip Tipp
Damit dein Server in der Serverliste angezeigt wird, muss die Option **Serverliste** aktiviert sein. Siehe [Serverliste aktivieren](serverliste-aktivieren.md).
::::
