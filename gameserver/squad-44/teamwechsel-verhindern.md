# So verhinderst du den Teamwechsel auf deinem Squad 44 Server

Du kannst verhindern, dass Spieler das Team wechseln, sobald sie sich einem Team angeschlossen haben.

1. <b>Verwaltung öffnen</b><br>
   Öffne die Verwaltung deines Servers.

2. <b>Einstellungen öffnen</b><br>
   Navigiere zu den **Einstellungen**.

3. <b>Teamwechsel verhindern anpassen</b><br>
   Setze das Feld **Teamwechsel verhindern** auf den gewünschten Wert:
   - `true` — Teamwechsel verhindern ist aktiviert
   - `false` — Teamwechsel verhindern ist deaktiviert

4. <b>Server neu starten</b><br>
   Speichere die Einstellung und starte deinen Server neu.
