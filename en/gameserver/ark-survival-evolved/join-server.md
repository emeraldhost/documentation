# How to Join Your ARK Survival Evolved Server

## Find connection details

:::: warning Important
To connect you need the **IP address** and the **Query Port** of your server. You can find the Query Port in the **Settings** of your server in the dashboard. Use the **Query Port**, not the game port!
::::

## Via Steam server browser

1. <b>Open Steam</b><br>
   Open the Steam client.

2. <b>Open server browser</b><br>
   Click on **View** in the top left and select **Game Servers**.

3. <b>Add server</b><br>
   Click on the **Favorites** tab and then on **+** at the bottom right.

4. <b>Enter server address</b><br>
   Enter the IP address and the **Query Port** of your server.

   :::: tip Example
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>Start ARK</b><br>
   Launch ARK Survival Evolved.

6. <b>Search for server</b><br>
   Select **Join ARK** in the main menu and filter by **Favorites** — your server should now appear in the list.

7. <b>Join the server</b><br>
   Select your server and click **Join**.

## Search by server name

1. <b>Start ARK</b><br>
   Launch ARK Survival Evolved.

2. <b>Open Join ARK</b><br>
   Select **Join ARK** in the main menu.

3. <b>Set filter</b><br>
   Set the session filter to **Unofficial**.

4. <b>Enter server name</b><br>
   Enter the name of your server in the **Name Filter** search field.

5. <b>Join the server</b><br>
   Select your server from the list and click **Join**.

## Via in-game console

1. <b>Start ARK</b><br>
   Launch ARK Survival Evolved and select **Join ARK** in the main menu.

2. <b>Open console</b><br>
   Press `Tab` to open the console.

3. <b>Connect</b><br>
   Enter the following command:

   ```
   open IP:Port
   ```

   :::: tip Example
   ```
   open 123.45.67.89:27015
   ```
   ::::

:::: info Note
For the in-game console, you must use the **game port** (not the Query Port).
::::
