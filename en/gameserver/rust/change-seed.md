# How to Change the Seed of Your Rust Server

The seed determines which world is generated on your server.

:::: warning Warning
Changing the seed on an existing save has no effect — it only applies when a new map is generated.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change seed</b><br>
   Enter the desired value in the **World Seed** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
