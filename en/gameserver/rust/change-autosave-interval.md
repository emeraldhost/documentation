# How to Change the Autosave Interval of Your Rust Server

The autosave interval determines how often your server automatically saves.

:::: info Note
Sets the interval for automatically saving the server in seconds. Default: 60.
::::

:::: tip Tip
Lower values save more often but may cause brief lag spikes.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change autosave interval</b><br>
   Enter the desired value in seconds in the **Autosave** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
