# How to Set the Server URL of Your Rust Server

The server URL lets you link to your website, which is shown to players in-game.

:::: info Note
The URL for your server. It is displayed when players click the 'Visit Website' button.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set server URL</b><br>
   Enter the desired value in the **Server URL** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
