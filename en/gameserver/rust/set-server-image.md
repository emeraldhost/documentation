# How to Set the Server Image of Your Rust Server

The server image is shown at the top of your server listing.

:::: info Note
The image shown at the top of your server listing. Expects an image URL.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set server image</b><br>
   Enter the desired image URL in the **Server Image** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
