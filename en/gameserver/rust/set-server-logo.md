# How to Set the Server Logo of Your Rust Server

The server logo is a circular image shown in the Rust+ app.

:::: info Note
The circular server logo for the Rust+ app. Expects an image URL.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set server logo</b><br>
   Enter the desired image URL in the **Server Logo** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
