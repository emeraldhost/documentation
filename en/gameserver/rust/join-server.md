# How to Join Your Rust Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via the F1 console

1. <b>Launch Rust</b><br>
   Start Rust via Steam.

2. <b>Open the console</b><br>
   In the main menu, press the `F1` key to open the console.

3. <b>Run the connect command</b><br>
   Enter the following command and press Enter:

   ```
   client.connect 123.45.67.89:28015
   ```

   Replace the IP address and port with those of your server.

## Via Steam favorites

Alternatively, you can add your server as a Steam favorite so it permanently appears in your list.

1. <b>Open Steam server browser</b><br>
   Open Steam and go to **View → Servers**.

2. <b>Open Favorites</b><br>
   Switch to the **Favorites** tab and click **Add a Server** at the bottom.

3. <b>Enter server address</b><br>
   Enter the IP address and port of your server and click **Add This Address to Favorites**.

4. <b>Join in Rust</b><br>
   Start Rust, click **Play Game** and switch to the **Favorites** tab. Select your server and click **Join**.
