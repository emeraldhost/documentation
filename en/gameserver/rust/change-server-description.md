# How to Change the Server Description of Your Rust Server

The server description is shown when players click on your server in the server browser.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change server description</b><br>
   Enter the desired value in the **Server Description** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
