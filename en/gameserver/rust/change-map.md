# How to Change the Map of Your Rust Server

The map setting determines which world Rust uses on your server.

:::: info Note
The world Rust should use. Common values are `Procedural Map`, `Barren`, or a specific map seed/name. Default: `Procedural Map`.
::::

:::: warning Warning
Only takes effect when a new map is generated.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change map</b><br>
   Enter the desired value in the **Map** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
