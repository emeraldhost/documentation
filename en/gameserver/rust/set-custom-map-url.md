# How to Set a Custom Map URL on Your Rust Server

A custom map URL lets you use your own map on your server.

:::: info Note
Lets you use a custom map (.map file) hosted online. Leave empty to use a procedural map.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set custom map URL</b><br>
   Enter the desired value in the **Custom Map URL** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
