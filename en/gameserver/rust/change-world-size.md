# How to Change the World Size of Your Rust Server

The world size determines how large the generated world on your server is.

:::: info Note
Typical values range from 1000 (small) to 6000 (large). Larger maps need more RAM and time to load.
::::

:::: warning Warning
Only takes effect when a new map is generated.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change world size</b><br>
   Enter the desired value in the **World Size** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
