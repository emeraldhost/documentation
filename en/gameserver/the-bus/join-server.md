# How to Join Your The Bus Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

:::: tip Server GUID
When the dedicated server starts, a **GUID** is displayed in the server console. You can use this as an alternative to the IP address to connect.
::::

## Via Direct Connect

1. <b>Launch The Bus</b><br>
   Launch The Bus and open the **Multiplayer** menu.

2. <b>Add server</b><br>
   Press the `Alt` key to add a new server.

3. <b>Enter server address</b><br>
   Enter the server GUID or the IP address with port.

   :::: tip Example
   ```
   123.45.67.89:7777
   ```
   ::::

4. <b>Connect</b><br>
   Connect to the server.
