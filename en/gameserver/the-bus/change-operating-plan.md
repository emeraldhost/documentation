# How to Change the Operating Plan on a The Bus Server

You can change the operating plan via the **admin menu** or by **command**.

## Change operating plan via admin menu

1. <b>Open admin menu</b><br>
   Open the pause menu and select the **Admin Menu**. You need Owner or Admin permissions.

2. <b>Select operating plan</b><br>
   Choose the desired operating plan from the available options.

## Change operating plan by command

Enter the following command in the in-game chat:

```
/operatingPlan <plan>
```

:::: tip Tip
This command requires Owner or Admin permissions.
::::
