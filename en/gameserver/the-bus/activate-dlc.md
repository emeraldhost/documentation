# How to Activate DLCs on a The Bus Server

You can activate or deactivate DLCs on your server by command.

## How to activate or deactivate a DLC

Enter the following command in the in-game chat:

```
/dlc
```

:::: tip Tip
This command requires Owner or Admin permissions.
::::
