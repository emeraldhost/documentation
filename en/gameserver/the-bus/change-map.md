# How to Change the Map on a The Bus Server

You can change the active map via the **admin menu** or by **command**.

## Change map via admin menu

1. <b>Open admin menu</b><br>
   Open the pause menu and select the **Admin Menu**. You need Owner or Admin permissions.

2. <b>Select map</b><br>
   Choose the desired map from the available options.

## Change map by command

Enter the following command in the in-game chat:

```
/map <map>
```

## Show available maps

To list all available maps:

```
/mapList
```

:::: tip Tip
These commands require Owner or Admin permissions.
::::
