# How to Change the Welcome Message of Your Wreckfest Server

The welcome message is shown to players when they join the server. It's useful for rules, hints or a greeting.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change welcome message</b><br>
   Enter the desired text in the **Welcome Message** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
