# How to Join Your Wreckfest Server

## Find connection details

:::: warning Important
To connect you need the **IP address** and the **Query Port** of your server. You can find the Query Port in the **Settings** of your server in the dashboard. Use the **Query Port**, not the game port!
::::

## Via the server browser

1. <b>Launch Wreckfest</b><br>
   Start Wreckfest.

2. <b>Open Multiplayer</b><br>
   Select **Multiplayer** in the main menu.

3. <b>Search for server</b><br>
   Search for your server's name in the search field or filter the list.

4. <b>Join the server</b><br>
   Select your server from the list and click **Join**. If your server is password-protected, enter the password and confirm.

## Via Steam favorites

Wreckfest uses the Steam server favorites. You can add your server there to find it more quickly.

1. <b>Open Steam</b><br>
   Open the Steam client.

2. <b>Open server browser</b><br>
   Click on **View** in the top left and select **Game Servers**.

3. <b>Add server</b><br>
   Click on the **Favorites** tab and then on **+** at the bottom right.

4. <b>Enter server address</b><br>
   Enter the IP address and the **Query Port** of your server.

   :::: tip Example
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>Launch Wreckfest</b><br>
   Start Wreckfest and open the **Multiplayer** area. Your server will be shown prominently.
