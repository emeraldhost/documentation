# FiveM

## What is FiveM?

FiveM is a multiplayer modification for Grand Theft Auto V that allows players to host and run their own custom servers with unique game modes, scripts and maps. FiveM was officially acquired by Rockstar Games in 2023 and has since become part of the GTA V platform.

FiveM is especially popular for **roleplay servers**, where players take on fictional roles — as police officers, mechanics, doctors, criminals and much more. There are also racing, drift, PvP and freeroam servers. The variety of server types and custom content is virtually unlimited.

Server owners can write their own scripts and frameworks using Lua, JavaScript and C#, or use existing frameworks like ESX or QBCore. With **txAdmin**, servers can be conveniently managed via a web interface.

FiveM is free to use for PC players who own GTA V on Steam, Epic Games or via the Rockstar Games Launcher.
