# How to Kick and Ban Players on Your RuneScape Dragonwilds Server

You can kick and ban players using the in-game admin menu. The admin password is required.

## Kick or ban a player

1. <b>Open the pause menu</b><br>
   Press the pause button to open the menu.

2. <b>Open settings</b><br>
   Navigate to **Settings** in the pause menu.

3. <b>Open Server Management</b><br>
   Select **Server Management**. Enter the admin password if prompted.

4. <b>Kick or ban the player</b><br>
   Select the player from the list and click the corresponding option to kick or ban them.

:::: info Note
You need the admin password to access the server management options.
::::
