# How to Set the Owner ID on Your RuneScape: Dragonwilds Server

The Owner ID is your Epic Games ID and is required for your server to start.

:::: tip Tip
Here you can find a guide on how to find your [Epic Games ID](../epicgamesid-find-out.md). You can also find your ID directly in-game under **Settings** at the bottom under **My Player ID**.
::::

## Enter Owner ID

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter Owner ID</b><br>
   Enter your Player ID in the **Owner ID** field.

4. <b>Start the server</b><br>
   Save the setting and start your server.

:::: danger Important
The server will **not start** until an Owner ID is set.
::::
