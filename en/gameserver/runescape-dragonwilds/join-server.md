# How to Join Your RuneScape: Dragonwilds Server

You can join your server via the world name in-game.

1. <b>Launch the game</b><br>
   Start RuneScape: Dragonwilds.

2. <b>Open worlds</b><br>
   Open the **Worlds** screen and select the **Public** tab.

3. <b>Search for server</b><br>
   Open the **Search** and enter the exact world name of your server.

   :::: warning Warning
   The world name is **case-sensitive**. It must match exactly.
   ::::

4. <b>Join the server</b><br>
   Wait a few seconds for your server to appear in the list and join.

:::: info Note
You can find the world name in the **Settings** of your server in the dashboard.
::::

:::: tip Troubleshooting
If your server cannot be found:
- Make sure the [Owner ID](set-owner-id.md) is set
- Make sure the world name matches exactly
- Check that your game and the server are on the same version
- Wait a few minutes after starting the server before searching
::::
