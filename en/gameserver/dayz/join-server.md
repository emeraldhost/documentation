# How to Join Your DayZ Server

You can join your server via the Steam server browser or directly via DayZ.

## Via Steam server browser

1. <b>Open Steam</b><br>
   Open the Steam client.

2. <b>Open server browser</b><br>
   Click on **View** in the top left and select **Game Servers**.

3. <b>Add server</b><br>
   Click on the **Favorites** tab and then on **+**.

4. <b>Enter server address</b><br>
   Enter the IP address and the **Query Port** of your server.

   :::: tip Example
   ```
   123.45.67.89:27016
   ```
   ::::

5. <b>Connect</b><br>
   Select the server and click **Connect**.

## Via DayZ in-game

1. <b>Start DayZ</b><br>
   Launch DayZ and select **Play** in the main menu.

2. <b>Open Community Servers</b><br>
   Select the **Community Servers** tab.

3. <b>Search for server</b><br>
   Search for your server by name or filter by IP address.

4. <b>Join the server</b><br>
   Select your server and click **Play**.

:::: warning Important
To connect you need the **Query Port**, **not** the game port. You can find the Query Port in the **Settings** of your server in the dashboard.
::::
