# How to Add an Admin to Your DayZ Server

You can log in as admin on your server by setting an admin password in the dashboard.

## Set admin password

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set admin password</b><br>
   Enter your desired password in the **Admin Password** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

## Log in as admin

1. <b>Join the server</b><br>
   Join your DayZ server.

2. <b>Open chat</b><br>
   Press `Enter` to open the chat.

3. <b>Admin login</b><br>
   Enter the following command:

   ```
   #login YourAdminPassword
   ```

:::: info Note
The admin login is only valid for the current session. After reconnecting, you need to enter the command again.
::::

:::: tip Tip
Only share the admin password with trusted players. Anyone with the password can grant themselves admin rights.
::::
