# How to Enable or Disable VoN on Your DayZ Server

With **Voice over Network (VoN)**, players can communicate with other players in-game via their microphone. You can enable or disable the feature via the dashboard.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Toggle VoN</b><br>
   Enable or disable the **Disable VoN** option:

   - **Enabled**: Players cannot speak via the game
   - **Disabled**: Players can communicate with each other via VoN

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
VoN only works within a certain range — nearby players can hear each other, distant players cannot. This creates a realistic communication experience.
::::
