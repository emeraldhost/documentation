# How to Join Your 7 Days to Die Server

You can join your server via the game or the Steam server browser.

## Via the game

1. <b>Launch the game</b><br>
   Launch 7 Days to Die.

2. <b>Join a game</b><br>
   Click on **Join a Game**.

3. <b>Connect to IP</b><br>
   Click on **Connect to IP** and enter the IP address and port of your server.

   :::: tip Example
   ```
   123.45.67.89:26900
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect**.

## Via Steam server browser

1. <b>Open Steam</b><br>
   Open the Steam client.

2. <b>Open server browser</b><br>
   Navigate to **View → Game Servers**.

3. <b>Add server</b><br>
   Click on the **Favorites** tab and then **Add Server**.

4. <b>Enter IP</b><br>
   Enter `IP:Port` of your server (e.g. `123.45.67.89:26900`).

5. <b>Connect</b><br>
   Select the server and click **Connect**.

:::: info Note
The default port is `26900`. You can find the IP address and port in the dashboard of your server.
::::
