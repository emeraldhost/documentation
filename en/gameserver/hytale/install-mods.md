# How to Install Mods on a Hytale Server

:::: tip Note
Stop your server before installing mods, otherwise they will not load correctly.
::::

## How to Install Mods

1. <b>Stop the Server</b><br>
   Stop your server via the management panel.

2. <b>Download the Mod</b><br>
   Download the desired mod as a `.jar` or `.zip` file.

3. <b>Upload the Mod</b><br>
   Connect to your server via [SFTP](../sftp-verbindung-herstellen.md) and upload the mod file to the `mods/` folder.

4. <b>Start the Server</b><br>
   Start your server to load the mod.

## How to Remove Mods

1. <b>Stop the Server</b><br>
   Stop your server via the management panel.

2. <b>Delete the Mod</b><br>
   Connect to your server via [SFTP](../sftp-verbindung-herstellen.md) and delete the mod file from the `mods/` folder.

3. <b>Start the Server</b><br>
   Start your server.

:::: warning Warning
Hytale is in Early Access. Mods may cause stability issues. Create a backup of your server before installation.
::::
