# How to Add an Admin on Your Conan Exiles Server

## Set admin password

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set admin password</b><br>
   Enter the desired password in the **Admin Password** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

## Log in as admin

1. <b>Join the server</b><br>
   Join your server.

2. <b>Open settings</b><br>
   Open the **Settings** in-game and navigate to **Server Settings**.

3. <b>Enter admin password</b><br>
   Enter your admin password in the corresponding field.

4. <b>Activate admin</b><br>
   Click **Make Me Admin** to log in as admin.

:::: info Note
The admin login only lasts for the current session. After reconnecting you need to log in as admin again.
::::

:::: tip Tip
Only share the admin password with trusted players. Anyone with the password can grant themselves admin rights.
::::
