# How to Enable or Disable BattlEye on Your Conan Exiles Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure BattlEye</b><br>
   Enter the desired value in the **BattlEye** field:

   | Value | Description |
   |-------|-------------|
   | `1` | BattlEye enabled |
   | `0` | BattlEye disabled |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
When BattlEye is enabled, all players must have BattlEye installed to join the server.
::::
