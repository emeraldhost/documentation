# How to Change the Server Region on Your Conan Exiles Server

The server region affects how your server is filtered in the server list. Choose a region to help players find a server near them.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Select region</b><br>
   Enter the desired value in the **Server Region** field:

   | Value | Region |
   |-------|--------|
   | `0` | Europe (default) |
   | `1` | North America |
   | `2` | Asia |
   | `3` | Australia |
   | `4` | South America |
   | `5` | Japan |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
