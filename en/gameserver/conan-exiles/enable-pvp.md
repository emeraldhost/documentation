# How to Enable or Disable PvP on Your Conan Exiles Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure PvP</b><br>
   Enter the desired value in the **PvP** field:

   | Value | Description |
   |-------|-------------|
   | `true` | PvP enabled — players can attack each other |
   | `false` | PvP disabled — players cannot attack each other (PvE) |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
