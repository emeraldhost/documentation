# How to Change the Server Community on Your Conan Exiles Server

The community affects how your server is filtered in the server list. Choose a community to help players find a server that matches their playstyle.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Select community</b><br>
   Enter the desired value in the **Server Community** field:

   | Value | Description |
   |-------|-------------|
   | `Purist` | Standard gameplay experience (default) |
   | `Relaxed` | Relaxed gameplay experience |
   | `HardCore` | Challenging gameplay experience |
   | `RolePlaying` | Roleplay-focused |
   | `Experimental` | Experimental settings |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
