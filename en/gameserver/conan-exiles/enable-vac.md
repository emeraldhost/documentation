# How to Enable or Disable VAC on Your Conan Exiles Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure VAC</b><br>
   Enter the desired value in the **VAC** field:

   | Value | Description |
   |-------|-------------|
   | `1` | VAC (Valve Anti-Cheat) enabled |
   | `0` | VAC disabled |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
