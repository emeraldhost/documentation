# How to Join Your SCP: Secret Laboratory Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Launch SCP: Secret Laboratory</b><br>
   Start SCP: Secret Laboratory via Steam.

2. <b>Open Servers</b><br>
   Click **Servers** in the main menu.

3. <b>Direct Connect</b><br>
   Select the **Direct Connect** option and enter the IP address and port of your server:

   :::: tip Example
   ```
   123.45.67.89:7777
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect** to join the server.

## Via Favorites

Alternatively, you can add your server as a Steam favorite so it permanently appears in your list.

1. <b>Open Steam</b><br>
   Open Steam and click **View → Servers** at the top.

2. <b>Open Favorites</b><br>
   Switch to the **Favorites** tab and click **Add a Server** at the bottom.

3. <b>Enter server address</b><br>
   Enter the IP address and port of your server and click **Add This Address to Favorites**.

4. <b>Join</b><br>
   Select your server from the favorites and click **Connect**.
