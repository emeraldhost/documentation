# RAGE:MP

## What is RAGE:MP?

RAGE:MP (RAGE Multiplayer) is an alternative multiplayer modification for Grand Theft Auto V. It allows players to create and host custom servers with unique game modes, vehicles and maps.

Developers can use C#, JavaScript and C++ to program custom gamemodes, creating unique experiences — from roleplay servers to racing and entirely custom game modes. RAGE:MP runs completely standalone and does not rely on any third-party mods.

RAGE:MP supports an unlimited number of players and provides a powerful API for server-side and client-side scripts. The game is available on PC.
