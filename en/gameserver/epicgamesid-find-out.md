# How to Find Your Epic Games ID

## Find out Epic Games ID via the web browser

1. <b>Open website</b><br>
   Go to the [Epic Games website](https://www.epicgames.com/site/home).

2. <b>Sign in</b><br>
   Click on "Sign in" and log in with your Epic Games account.

3. <b>Find ID</b><br>
   Click on "Account", then you will find your Epic Games ID under "Account information".

## Find out Epic Games ID via the launcher file

1. <b>Open folder</b><br>
   Press the `Windows key + R`, enter `AppData` and press the `Enter key`.

2. <b>Navigate to folder</b><br>
   Navigate to the following folder:

    ```
    Local/EpicGamesLauncher/Saved/Data
    ```

3. <b>Read ID</b><br>
   You should now see two `.dat` files. The Epic Games ID corresponds to the file name.
