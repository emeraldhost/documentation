# How to Join Your Renown Server

1. <b>Launch Renown</b><br>
   Start Renown.

2. <b>Select Play</b><br>
   Click **Play** in the main menu.

3. <b>Select region</b><br>
   Choose your region.

4. <b>Show all</b><br>
   Click **All** to show all available servers.

5. <b>Search for server</b><br>
   Search for your server's name in the search field.

6. <b>Join the server</b><br>
   Select your server from the list and click **Join**. If your server is password-protected, enter the password and confirm.

:::: tip Tip
For your server to appear in the public server list, [Public Server](change-public-server.md) must be set to `True` and [Private Session](change-private-session.md) must be set to `False`.
::::
