# How to Adjust the Map Rotation on Your Renown Server

The map rotation defines which maps are rotated during gameplay and in which order.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust map rotation</b><br>
   Enter the desired maps in the **Map Rotation** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
To make the server always load the first map from the rotation on startup, you can additionally enable [Use First Map](enable-use-first-map.md).
::::
