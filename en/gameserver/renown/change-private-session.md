# How to Enable Private Session on Your Renown Server

This setting determines whether your server runs as a private session.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust Private Session</b><br>
   Set the **Private Session** field to the desired value:
   - `True` — private session is enabled
   - `False` — private session is disabled

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: warning Important
If **Private Session** is set to `False`, [Public Server](change-public-server.md) must be set to `True`. The two settings are mutually exclusive.
::::
