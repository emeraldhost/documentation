# How to Enable Use First Map on Your Renown Server

When enabled, your server always loads the first map from the [Map Rotation](change-map-rotation.md) on startup, instead of selecting one randomly or based on the last game state.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust Use First Map</b><br>
   Set the **Use First Map** field to the desired value:
   - `True` — use first map is enabled
   - `False` — use first map is disabled

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
