# How to Set the Admin Password on Your Renown Server

The admin password lets you log in as an admin in-game and run admin commands.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set admin password</b><br>
   Enter the desired password in the **Admin Password** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: warning Warning
Keep the admin password secret and only share it with people you want to grant full server access.
::::
