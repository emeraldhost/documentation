# How to Enable First Person Only on Your Renown Server

When enabled, only the first-person view is available on your server. Players cannot switch to third-person view.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust First Person Only</b><br>
   Set the **First Person Only** field to the desired value:
   - `1` — first person only is enabled
   - `0` — first person only is disabled

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
