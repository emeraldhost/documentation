# How to Change the Server Name on an Aloft Server

:::: info Info
The server name is important to find your Aloft server in the public server list.
::::

1. <b>Open dashboard</b><br>
   Open your dashboard and select your Aloft server.

2. <b>Go to settings</b><br>
   Navigate to the settings page.

3. <b>Find server name</b><br>
   Scroll down to "Server Name".

4. <b>Change name</b><br>
   Change the name to the desired server name.

5. <b>Restart server</b><br>
   Restart your server so that the changes are applied.

:::: warning Attention
The server name must not contain spaces.
::::
