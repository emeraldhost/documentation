# How to Join an Aloft Server

## Room code variant

:::: tip Tip
The room code is displayed in the console when the server is started: `Server Ready : 123456 ;` You can also find the room code here without going to the file browser.
::::

:::: warning Attention
The room code changes with every server restart. Alternatively, you can use the public server list.
::::

1. <b>Open dashboard</b><br>
   Open your dashboard and select your Aloft server.

2. <b>Open file browser</b><br>
   Go to the file browser.

3. <b>Open room code file</b><br>
   Open the file `ServerRoomCode.txt`.

4. <b>Copy room code</b><br>
   Copy the 6-digit code from the file.

5. <b>Start Aloft</b><br>
   Launch the game.

6. <b>Select multiplayer</b><br>
   Click on "MULTIPLAYER".

7. <b>Join game</b><br>
   Select "JOIN GAME".

8. <b>Enter room code</b><br>
   Enter your room code in the top right-hand corner.

9. <b>Confirm join</b><br>
   Click on "JOIN GAME".

## Public server list variant

:::: info Info
If you cannot find your server, make sure that the server region matches the region in the server list.
::::

1. <b>Open dashboard</b><br>
   Open your dashboard and select your Aloft server.

2. <b>Go to settings</b><br>
   Navigate to the settings page.

3. <b>Find server list</b><br>
   Scroll down to "Server list".

4. <b>Enable server list</b><br>
   Set the value to `true`.

5. <b>Start Aloft</b><br>
   Launch the game.

6. <b>Select multiplayer</b><br>
   Click on "MULTIPLAYER".

7. <b>Join game</b><br>
   Select "JOIN GAME".

8. <b>Find your server</b><br>
   Now you can find your Aloft server in the list.
