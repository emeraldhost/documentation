# How to Add an Admin on an Aloft Server

:::: info Info
If you want to add multiple admins, you can simply separate the Steam64IDs with a space (e.g. `11111111111111111 22222222222222222`).
::::

:::: tip Tip
Here you will find instructions on how to find out your [Steam64ID](../steamid64-find-out.md).
::::

1. <b>Open dashboard</b><br>
   Open your dashboard and select your Aloft server.

2. <b>Go to settings</b><br>
   Navigate to the settings page.

3. <b>Find admin field</b><br>
   Scroll down to "Admins".

4. <b>Enter Steam64ID</b><br>
   Enter your Steam64ID in the field.

5. <b>Restart server</b><br>
   Restart your server so that the changes are applied.
