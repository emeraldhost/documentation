# How to Enable or Disable Crossplay on Your Valheim Server

With crossplay, players from different platforms (Steam, Xbox) can play together.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure crossplay</b><br>
   Enter the desired value in the **Enable Crossplay** field:

   | Value | Description |
   |-------|-------------|
   | `1` | Crossplay enabled |
   | `0` | Crossplay disabled |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
