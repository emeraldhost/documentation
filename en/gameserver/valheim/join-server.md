# How to Join Your Valheim Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Search by server name

1. <b>Start Valheim</b><br>
   Launch Valheim and select your character.

2. <b>Open Join</b><br>
   Click on **Join** and select the **Community** tab.

3. <b>Search for server</b><br>
   Use the filter and search for the name of your server in the server list.

4. <b>Join the server</b><br>
   Select your server from the list and click **Connect**.

## Via Direct Connect

1. <b>Start Valheim</b><br>
   Launch Valheim and select your character.

2. <b>Open Join</b><br>
   Click on **Join**.

3. <b>Enter IP</b><br>
   Click on **Join IP** at the bottom and enter the IP address and port of your server.

   :::: tip Example
   ```
   123.45.67.89:2456
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect** to join the server.
