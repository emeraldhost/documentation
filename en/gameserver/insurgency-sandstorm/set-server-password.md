# How to Set a Password on Your Insurgency: Sandstorm Server

You can protect your server with a password so that only players with the password can join. This is especially useful when your server appears in the server list via the [server tokens](set-game-stats-token.md) but you still don't want random players to join.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set password</b><br>
   Enter the desired password in the **Server Password** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
To remove the password and make your server public again, clear the password field and save.
::::
