# How to Change the Server Name of Your Insurgency: Sandstorm Server

You can change the display name of your server that is shown in the server list.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change server name</b><br>
   Enter the desired name in the **Server Name** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Note
For your server to appear in the server list at all, a [Game Stats Token](set-game-stats-token.md) and a [GSL Token](set-gsl-token.md) must also be set.
::::
