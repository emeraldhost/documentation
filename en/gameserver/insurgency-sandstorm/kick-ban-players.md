# How to Kick or Ban Players on Your Insurgency: Sandstorm Server

As an admin you can kick or permanently ban disruptive or rule-breaking players directly in-game via the admin menu.

## Requirement

You need admin rights on the server. To learn how to add yourself as an admin, see [Add Admins](add-admins.md).

## Kick or ban a player

1. <b>Join the server</b><br>
   Join your server.

2. <b>Open the admin menu</b><br>
   Open the admin menu with the assigned key — by default the **Minus** key (`-`) on the numpad.

3. <b>Open the player list</b><br>
   Switch to the **Players** tab.

4. <b>Select a player</b><br>
   Select the desired player and click **Kick** or **Ban**.

:::: tip Tip
For self-moderation without fixed admins, you can additionally enable [player voting](configure-player-voting.md), which lets players remove problematic players by vote.
::::
