# How to Join Your Insurgency: Sandstorm Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via the community server browser

1. <b>Launch Insurgency: Sandstorm</b><br>
   Start Insurgency: Sandstorm.

2. <b>Open the server browser</b><br>
   Select **Play** in the main menu and then **Community Servers**.

3. <b>Search for server</b><br>
   Search for your server's name or filter the list, e.g. by map or game mode.

4. <b>Join the server</b><br>
   Select your server from the list and join it. If your server is password-protected, enter the password.

## Via the console (direct connect)

1. <b>Open the console</b><br>
   In the main menu, press the `~` (tilde) key to open the console.

2. <b>Enter the connect command</b><br>
   Enter the following command and replace the IP and port with your connection details:

   ```
   open IP-Address:Port
   ```

   :::: tip Example
   ```
   open 123.45.67.89:27102
   ```
   ::::

3. <b>Connect</b><br>
   Confirm with `Enter` to join the server.

:::: info Note
For your server to appear in the community server browser, a [Game Stats Token](set-game-stats-token.md) and a [GSL Token](set-gsl-token.md) must be set.
::::
