# How to Set the Game Stats Token on Your Insurgency: Sandstorm Server

The Game Stats Token is required for your server to appear in the server list and for statistics to be tracked.

## Generate the token

1. <b>Open the Game Stats page</b><br>
   Open [https://gamestats.sandstorm.game/](https://gamestats.sandstorm.game/) and sign in.

2. <b>Create a token</b><br>
   Create a new token for your server and copy it.

## Set the token

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter the token</b><br>
   Paste the token into the **Game Stats Token** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Note
A [GSL Token](set-gsl-token.md) is also required for your server to appear in the server list.
::::
