# How to Change the Map on Your Insurgency: Sandstorm Server

The map determines which map is loaded when the server starts (e.g. `Ministry`, `Refinery`, `Hideout`).

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change the map</b><br>
   Enter the name of the desired map in the **Map** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Note
The map must match the selected [scenario](change-scenario.md). A full list of maps and scenarios can be found in the official [Server Admin Guide](https://mod.io/g/insurgencysandstorm/r/server-admin-guide).
::::
