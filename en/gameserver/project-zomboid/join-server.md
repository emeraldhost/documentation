# How to Join Your Project Zomboid Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Start Project Zomboid</b><br>
   Launch Project Zomboid.

2. <b>Open Multiplayer</b><br>
   Click on **Multiplayer** in the main menu.

3. <b>Join server</b><br>
   Select **Join Server**.

4. <b>Enter server details</b><br>
   Enter a name, the IP address and the port of your server.

   :::: tip Example
   ```
   IP: 123.45.67.89
   Port: 16261
   ```
   ::::

5. <b>Enter account details</b><br>
   Enter your username and the server password if applicable.

6. <b>Connect</b><br>
   Click **Join Server** to connect to the server.
