# How to Enable or Disable Private Mode on Your BeamMP Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure private mode</b><br>
   Enter the desired value in the **Private** field:

   | Value | Description |
   |-------|-------------|
   | `true` | Server is private and not visible in the server list |
   | `false` | Server is public and visible in the server list (default) |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
