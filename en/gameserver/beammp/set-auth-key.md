# How to Set the Authentication Key on Your BeamMP Server

To start your BeamMP server, you need an **authentication key**.

1. <b>Create key</b><br>
   Create a key at the [BeamMP Keymaster](https://beammp.com/keymaster).

2. <b>Open dashboard</b><br>
   Open the dashboard of your server.

3. <b>Open settings</b><br>
   Navigate to the **Settings**.

4. <b>Enter key</b><br>
   Enter the key in the **Authentication Key** field.

5. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: danger Important
Never share your authentication key with other people.
::::
