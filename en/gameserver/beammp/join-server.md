# How to Join Your BeamMP Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Search by server name

1. <b>Start BeamMP launcher</b><br>
   Start the BeamMP launcher and open BeamNG.drive via the launcher.

2. <b>Open BeamMP menu</b><br>
   Click on **MULTIPLAYER** in the main menu.

3. <b>Search for server</b><br>
   Use the filter and search for the name of your server in the server list.

4. <b>Join the server</b><br>
   Select your server from the list and click **Connect**.

## Via Direct Connect

1. <b>Start BeamMP launcher</b><br>
   Start the BeamMP launcher and open BeamNG.drive via the launcher.

2. <b>Open BeamMP menu</b><br>
   Click on **MULTIPLAYER** in the main menu.

3. <b>Direct Connect</b><br>
   Select **Direct Connect** and enter the IP address and port of your server.

   :::: tip Example
   ```
   123.45.67.89:30814
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect** to join the server.
