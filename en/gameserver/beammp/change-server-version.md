# How to Change the Server Version on Your BeamMP Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter version</b><br>
   Enter the desired version in the **BeamMP Server Version** field.

   :::: tip Example
   ```
   v3.1.0
   ```
   ::::

   All available versions can be found on [GitHub](https://github.com/BeamMP/BeamMP-Server/releases). Enter `latest` to always use the newest version.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
Invalid versions will automatically be reset to the latest version.
::::
