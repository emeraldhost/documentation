# How to Kick and Ban Players on Your BeamMP Server

You can kick and ban players using the server console.

## Kick or ban players via the server console

1. <b>Open the server console</b><br>
   Open the console of your server via the dashboard.

2. <b>Kick or ban a player</b><br>
   Use one of the following commands:

   ```
   kick <Name>
   ```

   ```
   ban <Name>
   ```

:::: info Note
A kicked player can rejoin the server immediately. A banned player is permanently excluded from the server.
::::

## Command overview

| Command | Description |
|---------|-------------|
| `kick <Name>` | Kick a player from the server |
| `ban <Name>` | Permanently ban a player |
