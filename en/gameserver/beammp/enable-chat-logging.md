# How to Enable or Disable Chat Logging on Your BeamMP Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure chat logging</b><br>
   Enter the desired value in the **Chat Logging** field:

   | Value | Description |
   |-------|-------------|
   | `true` | Chat logging enabled |
   | `false` | Chat logging disabled |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
