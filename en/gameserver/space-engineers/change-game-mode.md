# How to Change the Game Mode of Your Space Engineers Server

The game mode determines which mode your server runs in.

- **Survival** — survival mode with resource management, power, oxygen and manufacturing.
- **Creative** — creative mode with unlimited resources and free building.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change game mode</b><br>
   Select the desired value (`Survival` or `Creative`) in the **Game Mode** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
