# How to Join Your Space Engineers Server

## Find connection details

:::: info Note
You can find the **IP address** and **port** of your server in the **dashboard** of your server under the overview. Use exactly this address to connect.
::::

## Via Direct Connect

1. <b>Start Space Engineers</b><br>
   Launch Space Engineers and select **Join Game** in the main menu.

2. <b>Open Direct Connect</b><br>
   Click **Direct Connect**. If a Quick Start popup appears, choose **No**.

3. <b>Enter the address</b><br>
   Enter the IP address and port of your server:

   ```
   123.45.67.89:27016
   ```

4. <b>Connect</b><br>
   Click **Connect** to join the server.

## Via the server browser

1. <b>Start Space Engineers</b><br>
   Launch Space Engineers and select **Join Game** in the main menu.

2. <b>Find the server</b><br>
   Open the **Servers** tab and find your server in the list. Right-click it to add it to your **Favorites**.

3. <b>Join the server</b><br>
   Select your server and click **Connect**.

:::: info Note
The default Space Engineers port is `27016`. Your server may use a different port, however — always use the port shown in the dashboard.
::::

:::: tip Tip
For your server to appear publicly in the **Servers** tab it must be running and set to public. Private servers can only be reached via **Direct Connect**.
::::
