# How to Enable the Remote API on Your Space Engineers Server

The Remote API (VRage Remote API) allows your server to be managed remotely through an interface — for example to query the server status or control the server via external tools.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enable the Remote API</b><br>
   Set the **Remote API** option to `true` to enable it, or to `false` to disable it.

4. <b>Restart the server</b><br>
   Save the setting and restart your server for the change to take effect.

:::: info Note
The port for the Remote API is already set automatically — you do not need to configure it yourself.
::::
