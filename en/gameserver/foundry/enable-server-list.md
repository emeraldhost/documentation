# How to Enable the Server List on Your FOUNDRY Server

When the server list is enabled, your server is listed in the Steam server browser and can be found publicly.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enable server list</b><br>
   Enable the **Serverliste** toggle and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Info
When enabled, your server appears in the Steam server browser and can be found by other players there.
::::
