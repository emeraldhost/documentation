# How to Change the World Name of Your FOUNDRY Server

The world name determines which world folder is loaded on the server.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change world name</b><br>
   Enter the desired name in the **World Name** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
