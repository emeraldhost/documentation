# How to Change the World Seed of Your FOUNDRY Server

The world seed determines how a newly generated world looks. You can adjust the value in your server settings.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change seed</b><br>
   Enter the desired value in the **World Seed** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: warning Important
Changing the seed only affects newly generated worlds. Existing worlds keep their original seed.
::::
