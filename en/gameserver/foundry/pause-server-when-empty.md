# How to Automatically Pause Your FOUNDRY Server When It Is Empty

With this setting, your server pauses as soon as no player is connected. This saves CPU and RAM until someone joins again.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enable the option</b><br>
   Enable the **Pause Server When Empty** option and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Info
Pauses the server when no one is connected. This saves CPU and RAM until a player joins again.
::::
