# How to Join Your FOUNDRY Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via IP Direct Connect

1. <b>Launch FOUNDRY</b><br>
   Start FOUNDRY via Steam.

2. <b>Open Multiplayer</b><br>
   Click **Multiplayer** in the main menu.

3. <b>IP Direct Connect</b><br>
   Select the **IP Direct Connect** option and enter the IP address and port of your server:

   :::: tip Example
   ```
   123.45.67.89:3724
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect** to join the server.

## Via the Steam server browser

Public FOUNDRY servers can also be found through the Steam server browser.

1. <b>Open Steam server browser</b><br>
   Open Steam and go to **View → Servers**.

2. <b>Open Favorites</b><br>
   Switch to the **Favorites** tab and click **Add a Server** at the bottom.

3. <b>Enter server address</b><br>
   Enter the IP address and port of your server and confirm.

4. <b>Join</b><br>
   Select your server from the favorites and click **Connect**.
