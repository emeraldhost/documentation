# VEIN

## What is VEIN?

VEIN is a post-apocalyptic survival sandbox game set in an open world devastated by a zombie apocalypse. Society has collapsed and players are among the few survivors who must gather supplies, defend bases and rebuild civilization with friends.

The game combines crafting, base building, hunting, fishing and farming with intense combat against intelligent undead. Enemies can hear, see and smell you — and the more dangerous types react accordingly. Dynamic weather and random long-term events affect gameplay and keep things fresh.

Base building is a core element: players can build fortifications anywhere in the world to defend against invasions and other players. Nearly everything in the world is interactive — from mailboxes to cans you can throw at zombies.

VEIN supports both cooperative multiplayer and PvP. The game is available on PC via Steam.
