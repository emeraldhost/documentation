# How to Set the Server Owner on Your Astroneer Server

The server owner determines which Steam account receives administrator rights on the server. You need to enter both the Steam display name and the Steam64 ID.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter Server Owner Name</b><br>
   Enter the Steam display name of the owner in the **Server Owner Name** field.

4. <b>Enter Server Owner GUID</b><br>
   Enter the owner's [SteamID64](../steamid64-find-out.md) in the **Server Owner GUID** field.

5. <b>Restart the server</b><br>
   Save the settings and restart your server.

:::: info Note
Admin rights are assigned to the first person who connects with the specified name.
::::
