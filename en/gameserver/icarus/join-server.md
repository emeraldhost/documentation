# How to Join Your Icarus Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Launch Icarus</b><br>
   Start Icarus via Steam.

2. <b>Select Play</b><br>
   Click **Play** in the main menu.

3. <b>Choose character</b><br>
   Select your character.

4. <b>Select Join</b><br>
   Click **Join** and choose the **Dedicated Servers** section.

5. <b>Direct Connect</b><br>
   Select the **Direct Connect** option and enter the IP address and port of your server:

   :::: tip Example
   ```
   123.45.67.89:17777
   ```
   ::::

6. <b>Connect</b><br>
   Click **Connect** to join the server.

## Via the server list

Alternatively, you can find your server through the server list.

1. <b>Open Dedicated Servers</b><br>
   In the Join menu, select **Dedicated Servers (Beta)** in the top right.

2. <b>Search for the server</b><br>
   Search for the name you gave your server.

3. <b>Join the server</b><br>
   Select your server from the list and click **Join**.

:::: warning Warning
Your Icarus client version must match the server version. Check Steam for updates if joining fails.
::::
