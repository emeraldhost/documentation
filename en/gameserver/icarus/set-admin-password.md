# How to Set an Admin Password on Your Icarus Server

The admin password is required to log in as an administrator in-game.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set admin password</b><br>
   Enter the desired admin password in the **Admin Password** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
Only share the admin password with trusted players. Anyone with the password can grant themselves administrator privileges.
::::
