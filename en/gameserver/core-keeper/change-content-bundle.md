# How to Change the Content Bundle on Your Core Keeper Server

With content bundles you can enable additional content on your server.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter content bundle</b><br>
   Enter the desired bundle name in the **Content Bundle** field. To enable multiple bundles, separate the names with commas without spaces.

   :::: tip Example
   ```
   GiantCicadaBossDungeon,NatureBiomeCicadas,GuaranteedOases
   ```
   ::::

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
If the field is left empty, the server loads the default bundle.
::::
