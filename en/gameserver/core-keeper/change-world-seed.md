# How to Change the World Seed on Your Core Keeper Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter world seed</b><br>
   Enter the desired seed in the **World Seed** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
The seed is only used when creating a new world. Changing it on an existing world has no effect.
::::
