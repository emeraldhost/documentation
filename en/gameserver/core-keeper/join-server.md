# How to Join Your Core Keeper Server

## Find connection details

:::: info Note
You can find the **Game ID** of your server in the **dashboard** of your server under the overview or in the server console at startup.
::::

## Via Game ID

1. <b>Start Core Keeper</b><br>
   Launch Core Keeper.

2. <b>Select character</b><br>
   Select your character.

3. <b>Join Co-op</b><br>
   Click on **Co-op** and select **Join Game**.

4. <b>Enter Game ID</b><br>
   Enter the **Game ID** of your server in the input field.

5. <b>Connect</b><br>
   Click **Join** to connect to the server.
