# How to Change the Game ID on Your Core Keeper Server

The Game ID is the unique identifier of your server. Players need it to join the server.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter Game ID</b><br>
   Enter the desired ID in the **Game ID** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
The Game ID must be at least **28 characters** long and alphanumeric. The following characters are **not** allowed: `Y`, `y`, `x`, `0`, `O`. If the field is empty or invalid, a new ID will be generated automatically on startup.
::::
