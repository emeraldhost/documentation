# How to Change the World Index on Your Core Keeper Server

The world index determines which world is loaded. It must match the number of your world file (e.g. `0` for `0.world.gzip`).

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter world index</b><br>
   Enter the desired number in the **World Index** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: tip Example
If your world file is named `3.world.gzip`, the world index must be `3`.
::::
