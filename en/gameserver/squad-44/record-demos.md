# How to Enable Demo Recording on Your Squad 44 Server

Demos are recordings of matches that you can review later or use to investigate incidents.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust Record Demos</b><br>
   Set the **Record Demos** field to the desired value:
   - `true` — demo recording is enabled
   - `false` — demo recording is disabled

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
Demo files can take up significant disk space over time. Delete or download old demos regularly if your storage is running low.
::::
