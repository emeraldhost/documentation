# How to Enable the Server List on Your Squad 44 Server

This setting determines whether your server is shown in the public Custom Browser or not.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust Server List</b><br>
   Set the **Server List** field to the desired value:
   - `true` — server list is enabled (your server is shown in the Custom Browser)
   - `false` — server list is disabled (your server is hidden)

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: info Note
If the server list is disabled, players cannot find your server through the Custom Browser. They will need to reach the server through other means.
::::
