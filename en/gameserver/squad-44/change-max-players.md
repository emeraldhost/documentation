# How to Change the Maximum Players and Reserved Slots on Your Squad 44 Server

You can adjust the maximum number of concurrent players as well as the number of reserved slots for admins and whitelisted players.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust values</b><br>
   Enter the desired values in the following fields:
   - **Maximum Players** — the maximum number of concurrent players.
   - **Reserved Slots** — the number of slots reserved for admins and whitelisted players.

4. <b>Restart the server</b><br>
   Save the settings and restart your server.

:::: info Note
Reserved slots are included in the total **Maximum Players** count. Example: 80 maximum players with 5 reserved slots means 75 regular players can join — the remaining 5 slots are available only to admins and whitelisted players.
::::
