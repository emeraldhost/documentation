# How to Join Your Squad 44 Server

Squad 44 does not offer a direct connection via IP address — you always join your server through the **Custom Browser** in-game.

## Prerequisite

Note down the **server name** of your server. You can change it via the dashboard under **Settings** — see [Change Server Name](change-server-name.md).

## Join the server

1. <b>Start Squad 44</b><br>
   Launch Squad 44 via Steam.

2. <b>Open server browser</b><br>
   Select **Server Browser** in the main menu.

3. <b>Open Custom Browser</b><br>
   Switch to the **Custom Browser** tab.

4. <b>Search for server</b><br>
   Enter your server's name in the search field.

5. <b>Join the server</b><br>
   Select your server from the list and click **Join Server**. If your server is password-protected, enter the password and confirm.

:::: tip Tip
For your server to appear in the server list, the **Server List** option must be enabled. See [Enable Server List](enable-server-list.md).
::::
