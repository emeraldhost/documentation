# How to Add Mods to Your Palworld Server

Since version 0.7.0, Palworld officially supports mods through its built-in mod system and the Steam Workshop.

:::: warning Server-side mods currently not available
According to Pocketpair, server-side mods only work on the **Windows** dedicated server. Our Palworld servers run on **Linux** — server-side mods are therefore currently not supported. As soon as Pocketpair adds Linux support, we will update this guide.
::::

## Using client-side mods

Many mods are purely client-side (for example visual, sound or UI tweaks) and work independently of the server. Each player installs these locally on their own PC:

- [Steam Workshop](https://steamcommunity.com/app/1623730/workshop/) — subscribe to the mod and it is loaded automatically in-game
- [Nexus Mods](https://www.nexusmods.com/games/palworld/mods) and [CurseForge](https://www.curseforge.com/palworld) — install according to the respective mod description

Whether a mod works client-side or requires a (Windows) server is stated in the respective mod description.

:::: warning Warning
Mod support in Palworld is considered experimental and can cause crashes or corrupted save data — use at your own risk. Before updating to version 1.0 it is recommended to completely remove all mod files — disabling them is not enough. Only re-add mods once their authors have updated them for 1.0.
::::
