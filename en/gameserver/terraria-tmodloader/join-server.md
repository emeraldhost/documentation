# How to Join Your Terraria: tModLoader Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Start tModLoader</b><br>
   Launch Terraria: tModLoader.

2. <b>Select character</b><br>
   Click on **Multiplayer** and select your character.

3. <b>Join via IP</b><br>
   Select **Join via IP**.

4. <b>Enter server details</b><br>
   Enter the IP address and port of your server.

   :::: tip Example
   ```
   IP: 123.45.67.89
   Port: 7777
   ```
   ::::

5. <b>Connect</b><br>
   Click **Accept** to join the server.

:::: warning Warning
Make sure you have the **same mods** installed as the server, otherwise you cannot join.
::::
