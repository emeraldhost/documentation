# How to Change the Game Mode on Your Counter-Strike 2 Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Select gamemode</b><br>
   Select the desired game mode from the **Gamemode** dropdown.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

## Available game modes

| Game Mode | Description |
|-----------|-------------|
| `competitive` | Classic 5v5 ranked match |
| `wingman` | 2v2 on smaller maps |
| `casual` | Relaxed gameplay with larger teams |
| `deathmatch` | Free deathmatch without rounds |
| `custom` | Custom game mode |

:::: tip Tip
You can change the default map via the **Map** field in the settings.
::::
