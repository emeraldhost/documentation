# How to Change the Savegame Name of Your Empyrion Server

You can set which save is loaded by your server.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change savegame name</b><br>
   Enter the desired value in the **Save Game** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Info
This value determines which savegame folder is loaded. Changing it loads a different save or creates a new one.
::::
