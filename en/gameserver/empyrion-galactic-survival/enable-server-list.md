# How to Enable the Server List on Your Empyrion Server

You can set whether your server is publicly listed in the server list.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enable server list</b><br>
   Enable the **Server List** toggle and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Info
When enabled, your server appears in the Steam server browser.
::::
