# How to Change the Server Description of Your Empyrion Server

You can customize your server's description that is shown to players in the server list.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change server description</b><br>
   Enter the desired text in the **Server Description** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
