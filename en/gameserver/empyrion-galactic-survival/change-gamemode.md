# How to Change the Gamemode of Your Empyrion Server

You can set which game mode your server runs in.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Change gamemode</b><br>
   Enter the desired value in the **Gamemode** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: info Info
Typical values are `Survival` or `Creative`.
::::
