# How to Join Your Empyrion: Galactic Survival Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct IP and Port

1. <b>Launch Empyrion</b><br>
   Start Empyrion: Galactic Survival via Steam.

2. <b>Open Join a Game</b><br>
   Click **Join a Game** in the main menu.

3. <b>Enter Direct IP and port</b><br>
   In the **Direct IP and port** field, enter the IP address and port of your server:

   :::: tip Example
   ```
   123.45.67.89:30000
   ```
   ::::

4. <b>Connect</b><br>
   Click **Connect** to join the server.

## Via the Steam server browser

Alternatively, you can add your server via the Steam server browser.

1. <b>Open Steam server browser</b><br>
   Open Steam and go to **View → Servers**.

2. <b>Open Favorites</b><br>
   Switch to the **Favorites** tab and click **Add a Server** at the bottom.

3. <b>Enter server address</b><br>
   Enter the IP address and query port of your server and confirm.

4. <b>Join</b><br>
   Select your server from the favorites and click **Connect**.
