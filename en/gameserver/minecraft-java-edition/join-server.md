# How to Join Your Minecraft Java Edition Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Start Minecraft</b><br>
   Launch Minecraft Java Edition.

2. <b>Open Multiplayer</b><br>
   Click on **Multiplayer** in the main menu.

3. <b>Add server</b><br>
   Click on **Add Server**.

4. <b>Enter server details</b><br>
   Enter a name for your server and the server address in the format `IP:Port`.

   :::: tip Example
   ```
   123.45.67.89:25565
   ```
   ::::

5. <b>Done</b><br>
   Click **Done**. Your server will now appear in the list and you can join it.

:::: tip Tip
If you have connected a domain to your server, you can use it instead of the IP address. See [Connect a domain to a Minecraft server](../../domains/dns-records/minecraft-server.md).
::::
