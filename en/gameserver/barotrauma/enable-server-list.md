# How to Enable or Disable the Server List on Your Barotrauma Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure server list</b><br>
   Enter the desired value in the **Server List** field:

   | Value | Description |
   |-------|-------------|
   | `True` | Server is shown in the public server list |
   | `False` | Server is not shown in the server list |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
