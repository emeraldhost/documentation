# How to Change the Game Mode on Your Barotrauma Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Select game mode</b><br>
   Enter the desired value in the **Gamemode** field:

   | Value | Description |
   |-------|-------------|
   | `Sandbox` | Free play without restrictions |
   | `Mission` | Play individual missions |
   | `Campaign` | Campaign mode with progression |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
