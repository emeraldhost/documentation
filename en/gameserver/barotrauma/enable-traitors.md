# How to Enable Traitors on Your Barotrauma Server

You can enable the traitor system so that players may be secretly assigned as traitors during rounds.

:::: info Note
You need admin rights on the server to change these settings.
::::

1. <b>Open Game Mode Settings</b><br>
   While in the server lobby, open the **Game Mode Settings** menu.

2. <b>Open Traitor Settings</b><br>
   Navigate to the **Traitor Settings** section.

3. <b>Set Traitor Probability</b><br>
   Set **Traitor Probability** to a value greater than **0**. A higher value means traitors appear more frequently.

4. <b>Set Max Danger Level</b><br>
   Set the **Max Danger Level** to control how difficult traitor objectives can be.

5. <b>Set minimum players</b><br>
   Set the minimum number of players required for a traitor to be assigned. This prevents traitors from appearing in very small crews.

6. <b>Start the round</b><br>
   Click **Start** to apply the settings and begin the round with traitors enabled.

:::: tip Tip
Start with a low traitor probability and increase it over time to find the right balance for your group.
::::
