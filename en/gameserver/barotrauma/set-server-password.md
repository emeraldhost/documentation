# How to Set a Server Password on Your Barotrauma Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set server password</b><br>
   Enter the desired password in the **Server Password** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: tip Tip
To remove the password, clear the field and save.
::::
