# How to Set an Admin Password on Your Soulmask Server

You can set an admin password so that players can grant themselves admin rights ingame.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set admin password</b><br>
   Enter the desired admin password and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
Choose a strong password and only share it with trusted players.
::::
