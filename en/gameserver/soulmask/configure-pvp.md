# How to Configure PvP and PvE on Your Soulmask Server

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter game mode</b><br>
   Enter the desired mode in the **Game Mode** field:

   | Value | Description |
   |-------|-------------|
   | `pve` | Players cannot attack each other |
   | `pvp` | Players can attack each other at any time |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
