# How to Join Your Soulmask Server

## Find connection details

:::: warning Important
To connect you need the **IP address** and the **Query Port** of your server. You can find the Query Port in the **Settings** of your server in the dashboard. Use the **Query Port**, not the game port!
::::

## Via Steam server browser

1. <b>Open Steam</b><br>
   Open the Steam client.

2. <b>Open server browser</b><br>
   Click on **View** in the top left and select **Game Servers**.

3. <b>Add server</b><br>
   Click on the **Favorites** tab and then on **+** at the bottom right.

4. <b>Enter server address</b><br>
   Enter the IP address and the **Query Port** of your server.

   :::: tip Example
   ```
   123.45.67.89:27015
   ```
   ::::

5. <b>Start Soulmask</b><br>
   Launch Soulmask.

6. <b>Search for server</b><br>
   Your server should now appear in the server list under **Favorites**.

7. <b>Join the server</b><br>
   Select your server and click **Join**.

## Search by server name

1. <b>Start Soulmask</b><br>
   Launch Soulmask.

2. <b>Select Online Game</b><br>
   Select **Online Game** in the main menu.

3. <b>Select Private Server</b><br>
   Select the **Private Server** tab.

4. <b>Search for server</b><br>
   Search for the name of your server in the search field at the top right.

5. <b>Join the server</b><br>
   Select your server from the list and join.

## Via Direct Connect

1. <b>Start Soulmask</b><br>
   Launch Soulmask.

2. <b>Select Online Game</b><br>
   Select **Online Game** in the main menu.

3. <b>Direct Connect</b><br>
   Click on **Connect directly to server** at the bottom.

4. <b>Enter server details</b><br>
   Enter the **IP address**, the **port** and optionally the **server password**.

5. <b>Connect</b><br>
   Click **Connect** to join the server.
