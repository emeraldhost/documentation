# How to Change the Map on Your Soulmask Server

You can switch between the available maps.

:::: warning Warning
For the Shifting Sands map you need the **Shifting Sands DLC**. All players must also own the DLC. When switching maps, players will be prompted to create a new character.
::::

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Select map</b><br>
   Change the map setting to the desired value:

   | Value | Map |
   |-------|-----|
   | `Level01_Main` | Cloud Mist Forest (default) |
   | `DLC_Level01_Main` | Shifting Sands (DLC) |

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: tip Tip
If you want to run both maps simultaneously and allow players to travel between them, you need a server cluster with two servers. See [Setup Shifting Sands](setup-shifting-sands.md).
::::
