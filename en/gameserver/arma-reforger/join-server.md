# How to Join Your Arma Reforger Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Search by server name

1. <b>Start Arma Reforger</b><br>
   Launch Arma Reforger.

2. <b>Open Multiplayer</b><br>
   Click on **Multiplayer** in the main menu.

3. <b>Search for server</b><br>
   Use the filter and search for the name of your server in the server list.

4. <b>Join the server</b><br>
   Select your server from the list and click **Join**.

## Via Direct Connect

1. <b>Start Arma Reforger</b><br>
   Launch Arma Reforger.

2. <b>Open Multiplayer</b><br>
   Click on **Multiplayer** in the main menu.

3. <b>Direct Join</b><br>
   Click on **Direct Join** and enter the IP address and port of your server.

   :::: tip Example
   ```
   123.45.67.89:2001
   ```
   ::::

4. <b>Connect</b><br>
   Click **Join** to connect to the server.
