# How to Set the Island ID on Your Windrose Server

The island ID links your server to a specific world. It can be found in the `WorldDescription.json` file. If no island ID is set, the most recent save in the `/R5/Saved/` directory is used automatically.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set island ID</b><br>
   Enter the desired island ID in the **Island ID** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
