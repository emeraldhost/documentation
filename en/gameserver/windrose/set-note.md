# How to Set a Note on Your Windrose Server

You can attach a note with additional information about your server. This helps in rare cases to distinguish your server from others when invite codes between two servers look similar.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set note</b><br>
   Enter the desired information in the **Note** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.
