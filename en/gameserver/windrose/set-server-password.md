# How to Set a Password on Your Windrose Server

You can protect your server with a password so that only players with the password can join.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enable password protection</b><br>
   Enable the **Server Password Protected** toggle.

4. <b>Set password</b><br>
   Enter the desired password in the **Server Password** field and save the setting.

5. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
To remove the password and make the server public again, disable the **Server Password Protected** toggle or leave the password field empty.
::::
