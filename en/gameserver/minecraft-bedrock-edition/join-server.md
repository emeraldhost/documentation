# How to Join Your Minecraft Bedrock Edition Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Start Minecraft</b><br>
   Launch Minecraft Bedrock Edition.

2. <b>Play</b><br>
   Click on **Play**.

3. <b>Open Servers tab</b><br>
   Select the **Servers** tab.

4. <b>Add server</b><br>
   Scroll down and click on **Add Server**.

5. <b>Enter server details</b><br>
   Enter a name for your server, the IP address and the port.

   :::: tip Example
   ```
   IP: 123.45.67.89
   Port: 19132
   ```
   ::::

6. <b>Save</b><br>
   Click **Save**. Your server will now appear in the list and you can join it.

## On console (Xbox, PlayStation, Switch)

Consoles do not allow you to add custom server IPs directly. Using a DNS trick (BedrockConnect) you can redirect the "Featured Servers" list so that you can enter your own IP.

:::: warning Warning
This changes your DNS settings. Note down your current values beforehand in case you want to restore them later.
::::

### DNS addresses

| Console | Primary | Secondary |
|---------|---------|-----------|
| Xbox One / Series X\|S | `104.238.130.180` | `8.8.8.8` |
| PlayStation 4 / 5 | `45.55.68.52` | `8.8.8.8` |
| Nintendo Switch | `104.238.130.180` | `8.8.8.8` |

### Xbox

1. <b>Open network settings</b><br>
   Go to **Settings → General → Network settings → Advanced settings → DNS settings**.

2. <b>Select Manual</b><br>
   Choose **Manual** and enter the DNS addresses from the table.

3. <b>Save and restart</b><br>
   Save the setting and restart your Xbox.

### PlayStation

1. <b>Open network settings</b><br>
   Go to **Settings → Network → Settings → Set up internet connection**.

2. <b>Adjust connection</b><br>
   Select your network, open **Advanced Settings** and set **DNS Settings** to **Manual**.

3. <b>Enter DNS</b><br>
   Enter the DNS addresses from the table and confirm.

4. <b>Restart the PlayStation</b><br>
   Restart your console.

### Connect to the server

1. <b>Start Minecraft</b><br>
   Open Minecraft Bedrock Edition and switch to the **Servers** tab.

2. <b>Join a Featured Server</b><br>
   Click on any of the featured servers (e.g. Mineville, Lifeboat). You will be redirected to BedrockConnect instead.

3. <b>Enter your IP</b><br>
   Select **Connect to a Server** and enter the IP address and port of your server.
