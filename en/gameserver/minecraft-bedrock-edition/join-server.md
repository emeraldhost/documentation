# How to Join Your Minecraft Bedrock Edition Server

## Find connection details

:::: info Note
You can find the IP address and port of your server in the **dashboard** of your server under the overview.
::::

## Via Direct Connect

1. <b>Start Minecraft</b><br>
   Launch Minecraft Bedrock Edition.

2. <b>Play</b><br>
   Click on **Play**.

3. <b>Open Servers tab</b><br>
   Select the **Servers** tab.

4. <b>Add server</b><br>
   Scroll down and click on **Add Server**.

5. <b>Enter server details</b><br>
   Enter a name for your server, the IP address and the port.

   :::: tip Example
   ```
   IP: 123.45.67.89
   Port: 19132
   ```
   ::::

6. <b>Save</b><br>
   Click **Save**. Your server will now appear in the list and you can join it.
