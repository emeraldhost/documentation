# How to Increase the Slots on Your Minecraft Bedrock Edition Server

1. <b>Open dashboard</b><br>
   Open your dashboard and select your Minecraft Bedrock Edition server.

2. <b>Open file browser</b><br>
   Go to the file browser.

3. <b>Open config</b><br>
   Open the file ```server.properties```.

4. <b>Find entry</b><br>
   Look for the following entry:

    ```
    max-players=10
    ```

5. <b>Change value</b><br>
   Change the value to the desired number of players.

:::: warning Warning
Important: Make sure that your server has enough resources (such as RAM and CPU) to support the increased number of players.
::::
