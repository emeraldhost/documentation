# How to Set the License Key on Your FiveM Server

To start your FiveM server correctly, you need to set a **license key** in the settings. Without this key, your server will not start.

## Generate license key

1. <b>Open Cfx.re Portal</b><br>
   Open the official [Cfx.re Portal](https://portal.cfx.re/).

2. <b>Sign in</b><br>
   Sign in with your Cfx.re account or create a new one.

3. <b>Generate key</b><br>
   Create a new license key for your server and copy it.

   :::: info Note
   When generating the key, you will be asked for the **IP address** of your server. You can find this in the dashboard of your server.
   ::::

## Set license key

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Enter license key</b><br>
   Enter the generated key in the **License Key** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.

:::: warning Warning
Never share your license key with other people. Each server requires its own unique key.
::::
