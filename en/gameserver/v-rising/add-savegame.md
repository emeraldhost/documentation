# Adding a V Rising server savegame

::: danger Warning
When you add a new savegame, **your current game progress will be overwritten** and **cannot be restored**.  
Be sure to create a **backup** if you want to reuse your current save game later.
:::

::: info Info
You can find a local save game in the following directory on your PC: ```C:\Users\<username>\AppData\LocalLow\Stunlock Studios\VRising\Saves```
:::

1. <strong>Open your dashboard and select your V Rising server.</strong>

2. <strong>Stop your V Rising server.</strong>

2. <strong>Navigate to the ```Saves/v1``` folder.</strong>

3. <strong>Open the folder for your save game.</strong>

4. <strong>Delete the existing files and upload the new files for your desired save game.</strong>

5. <strong>Restart your server for the changes to take effect.</strong>
