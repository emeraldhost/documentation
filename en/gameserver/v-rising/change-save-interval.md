# How to Change the Save Interval on Your V Rising Server

You can configure how often and how many automatic saves the server creates.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust save settings</b><br>
   Adjust the following fields:

   | Field | Description |
   |-------|-------------|
   | **Auto Save Interval** | Time in seconds between automatic saves |
   | **Auto Save Count** | How many save files are kept |

4. <b>Restart the server</b><br>
   Save the settings and restart your server.
