# How to Enable or Disable the Server List on Your V Rising Server

You can configure whether your server is shown in the Steam and/or Epic Games server list.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Configure server list</b><br>
   Adjust the following fields:

   | Field | Values | Description |
   |-------|--------|-------------|
   | **Steam Server List** | `true` / `false` | Show server in Steam server list |
   | **Epic Games Server List** | `true` / `false` | Show server in Epic Games server list |

4. <b>Restart the server</b><br>
   Save the settings and restart your server.
