# How to Change the Game Settings on Your V Rising Server

You can adjust the preset game settings via the dashboard.

1. <b>Open dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Adjust game settings</b><br>
   Select the desired preset in the **Preset Game Settings** field.

4. <b>Restart the server</b><br>
   Save the setting and restart your server.
