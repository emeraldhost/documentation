# Adding a V Rising Server Admin

::: info Info
If you want to add multiple admins, simply paste **multiple Steam64IDs** - each on a **new line**.
:::

::: tip Tip
Here you will find instructions on how to find your [Steam64ID](../steamid64-find-out.md).
:::

1. <strong>Open your dashboard and select your V Rising server.</strong>

2. <strong>Navigate to the ```save-data/Settings``` folder.</strong>

3. <strong>Open the file ```adminlist.txt```.</strong>

4. <strong>Add your Steam64ID in a new line.</strong>

5. <strong>Save the file and restart your server for the changes to take effect.</strong>
