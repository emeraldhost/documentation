# Adding a V Rising Server Admin

:::: info Info
If you want to add multiple admins, simply paste **multiple Steam64IDs** - each on a **new line**.
::::

:::: tip Tip
Here you will find instructions on how to find your [Steam64ID](../steamid64-find-out.md).
::::

1. <b>Open dashboard</b><br>
   Open your dashboard and select your V Rising server.

2. <b>Open settings folder</b><br>
   Navigate to the `save-data/Settings` folder.

3. <b>Open adminlist.txt</b><br>
   Open the file `adminlist.txt`.

4. <b>Add Steam64ID</b><br>
   Add your Steam64ID in a new line.

5. <b>Restart server</b><br>
   Save the file and restart your server for the changes to take effect.
