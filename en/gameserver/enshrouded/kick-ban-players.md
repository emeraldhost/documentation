# How to Kick and Ban Players on Your Enshrouded Server

In Enshrouded, you can kick players from the in-game player list. A permanent ban system is currently not available.

## Kick players via the player list

1. <b>Join the server</b><br>
   Join your Enshrouded server as host or admin.

2. <b>Open the player list</b><br>
   Open the player list in the game.

3. <b>Kick a player</b><br>
   Select the player and click **Kick** to remove them from the server.

:::: info Note
Enshrouded currently does not offer a permanent ban system. A kicked player can rejoin the server.
::::

## Restrict access via server password

If you want to permanently exclude certain players from your server, you can set a server password:

1. <b>Open the dashboard</b><br>
   Open the dashboard of your server.

2. <b>Open settings</b><br>
   Navigate to the **Settings**.

3. <b>Set a password</b><br>
   Enter a password in the **Server Password** field and save the setting.

4. <b>Restart the server</b><br>
   Restart your server for the change to take effect.

:::: tip Tip
Only share the password with players you trust. Change the password if you want to exclude a player.
::::
