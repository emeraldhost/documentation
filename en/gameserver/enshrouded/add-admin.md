# How to Add an Admin to Your Enshrouded Server

In Enshrouded, the first player who joins the server with the admin password becomes the host. You can set the admin password via the dashboard.

## Set the Admin Password

1. <b>Open the dashboard</b><br>
   Open the dashboard and select your Enshrouded server.

2. <b>Go to settings</b><br>
   Navigate to the settings of your server.

3. <b>Set admin password</b><br>
   Enter your desired password under `Admin Password`.

4. <b>Restart the server</b><br>
   Save the settings and restart the server.

## Join as Admin

1. <b>Join the server</b><br>
   Connect to your Enshrouded server.

2. <b>Enter admin password</b><br>
   Enter the previously set admin password when joining. The first player who enters the correct password becomes the host.

:::: info Note
The host has full management rights on the server. Only share the admin password with trusted players.
::::

:::: tip Tip
You can also change the admin password directly in the `enshrouded_server.json` file on the server. Connect to your server via [SFTP](../establish-sftp-connection.md) to do so.
::::
